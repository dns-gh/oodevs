


Here are installation instruction concerning ArcSDE for postgresql, which
should be installed before running install-sword-sde-db.bat

The install-sword-sde-db.bat scripts may replace ArcSDE Postinstall, since it:
1) creates a database in pgsql
2) add functions and table allowing to turn it into a geodatabase (relying on sde)
3) creates an SDE Windows service

It moreover performs the following steps for Sword Application:
4) creates a schema for sword application (crossbow plugin)
5) register sword tables as ArcSDE layers
