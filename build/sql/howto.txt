The install-sword-db.bat scripts:
1) creates sword role and a database in pgsql
2) add functions and table allowing to turn it into a geodatabase 

It moreover performs the following steps for Sword Application:
4) creates a schema for sword application (crossbow plugin)



Postgresql intallation
service SecondaryLogon must be started

1) Install

 - install-sword-db: create sword database & schema.
   Run the command with one argument or set the variables in set_sword_var.bat: the expected database name
 
2) Uninstall

 - uninstall-sword-db
