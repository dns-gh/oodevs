NIVEAU : Pion

ARME : Melee

MISSION : Reconnaitre

Définition TTA 106

Il s'agit, progressant en premier échelon de la compagnie ou isolément, de rechercher le renseignement d'ordre tactique ou technique, sur l'ennemi ou le terrain, sur un point ou dans une zone donnée, en engageant éventuellement le combat

Effets

Variantes et modalités d'exécution

Reconnaître un point :
  1. Déplacement vers l’objectif
  2. Observation à partir d'une position offrant une vue sur l'objectif
  3. Si l'objectif est occupé le pion engage le combat (cas non détaillé pour le moment), sinon le pion l'occupe
  4. Après un délai de reconnaissance, le point est reconnu et le pion l'occupe

Reconnaître un bloc urbain :
  1. Déplacement vers le bloc urbain
  2. Observation de l'extérieur du bloc urbain à partir d'une position offrant une vue sur l'objectif
  3. Après un délai d'observation, le pion entre dans le bloc urbain pour le reconnaitre.
  4. Après un délai de reconnaissance, le bloc urbain est reconnu.

Reconnaître un itinéraire :
  1. Parcourir l’itinéraire de point en point en les reconnaissant
  2. Remontée du renseignement pour chacun de ces points
  3. Si un point est occupé le pion engage le combat (cas non détaillé pour le moment), sinon le pion l'occupe

Reconnaître une zone :
  1. Déplacement vers la zone
  2. Observation de l'extérieur de la zone à partir d'une position offrant une vue sur l'objectif
  3. Après un délai d'observation, le pion se déplace dans la zone en reconnaissant tous les points et/ou tous les blocs urbains de celle-ci. Les blocs urbains ne seront reconnus que de l'extérieur
