NIVEAU : Pion débarqué

ARME : Security

MISSION : Trier les victimes

Définition TTA 106

La mise en place d'un Poste médical avancé permet de traiter les victimes lorsqu'elles sont nombreuses. Les victimes sont triées : UR pour les urgences relatives, UA pour les urgences absolues, décès lorsqu'elles sont mortes, impactées psychologiques. Ce tri permet d'envoyer les UR vers les hôpitaux plus éloignés pour ne pas congestionner les plus proches qui accueillent les UA. Le délai de traitement des UA est essentiel : l'objectif est d'éviter que les UA deviennent des morts.

Effets

Variantes et modalités d'exécution

Rejoindre la zone de tri des victimes :
  1. Rejoindre la position du PMA (possible que si la zone de tri est opérationnelle).

Trier les victimes :
  1. Le pion détermine pour chaque victime si elle est une UA, une UR, morte ou impactée psychologique.

Fin de mission :
  1. Retour sur le point de garage.
