ARME :GENIE

MISSION :Activer Obstacles

Objectif principal

Modalités d'exécution

Activer des Obstacles :
  1. Progresser vers le premier obstacle ;
  2. Activer l'obstacle si les capacités de l'unité le permettent, attendre un renforcement dans le cas contraire ;
  3. Valoriser l'obstacle si les capacités de l'unité le permettent ;
  4. Répéter les étapes pour les obstacles suivants.
