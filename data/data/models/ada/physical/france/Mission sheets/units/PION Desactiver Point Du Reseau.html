NIVEAU :Pion

ARME :

MISSION :Desactiver Point Du Reseau

Objectif principal

  1. Désactiver

Modalités d'exécution

  1. Se diriger vers le point du réseau cible ;
  2. Désactiver le noeud du réseau (production et/ou distribution);
  3. Rejoindre le point de regroupement une fois la réparation effectuée.

Schéma de modélisation



Paramètres obligatoires

Point de regroupement : Position à rejoindre une fois le point du réseau désactivé.



Reseau de ressource
Type de ressource : le types de ressource à rétablir pour le point de réseau sélectionné.

Paramètres optionnels

Direction Dangereuse

Limite gauche / Limite droite

Lignes de début et fin de mission [LDM/LFM]
