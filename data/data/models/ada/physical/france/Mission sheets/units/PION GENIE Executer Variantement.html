Objectif principal

  1. Construire

Modalités d'exécution

  1. Démonter le pont
  2. Rejoindre le nouveau site de franchissement
  3. Organiser le chantier, aménager les berges (le site doit avoir été reconnu)
  4. Construire le pont sur le nouveau site
  5. Exploiter : rester sur le site
  6. Fin sur ordre (démonter le pont)

Schéma de modélisation



Paramètres obligatoires

Site de Franchissemment : Nouveau site de franchissement a équiper et exploiter


Type pontage : Mode continu ou discontinu


Paramètres optionnels

Limite gauche /  limite droite
