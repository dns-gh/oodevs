Mission : Crowd - Get decontaminated

Context

People may be contaminated during a NBC incident. To avoid casulaties and deads, contaminated people should be decontaminated with appropriate means.
In SWORD, a crowd must go through a decontamination site object to get decontaminated. The decontamination process is effective if the decontamination site is operated by other units.

Parameters

Mandatory

Decontamination site

Decontamination site to use to get decontaminated.

Meeting position

Position to reach after being decontaminated.

Optional

Behavior
  • The agent moves toward the issued decontamination site. If an obstacle is encountered on the way to the objective, the crowd stops and a new temporary concentration is created. The movement is suspended until the obstacle is removed.
  • Upon reaching the decontamination site, the agent requests for decontamination. The report "request for decontamination" should be displayed. Then, the agents get decontaminated with a delay depending on:
    • the number of units that are operating the site (if the site is not operated, the agent stays next to it until the decontamination site get operated or another mission is issued),
    • the number of units asking for being decontaminated on the site (the agent needs to queue to get decontaminated; if the agent asked first for being decontaminated, it should be the first to get decontaminated).
    • the number of individuals to decontaminate: they are decontaminated one by one (delay can be very long, verify datas set in authoring tool)
  • The crowd agent may not get decontaminated if the issued decontamination site cannot decontaminate (see objects capacities). In that case, the crowd agent displays the message "Impossible to get decontaminated by issued decontamaintion plot" and the mission ends.
  • Once the agent gets decontaminated (the report "unit decontaminated" should be displayed), it moves toward the garage position and the mission ends.
