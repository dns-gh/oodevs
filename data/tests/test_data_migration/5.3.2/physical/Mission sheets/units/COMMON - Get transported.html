Mission : COMMON - Get transported

Context

To move to a position in order to be transported by other unit(s)
In Sword to be transported by another unit. This mission is used to coordinate the troops and vehicules transportation (see transportation capabilities in the authoring tool).

Parameters

Mandatory

Danger direction

Expected enemy direction.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Position

Embarkation position.

Behavior
  • The agent moves toward the given position (if a position is provided; otherwise the agent will stay put). If it is not reachable (an urban block inside which the agent cannot move, for instance), the agent moves as close as possible, otherwise it moves to the center of the position. Average speed depends on the physical capabilities, taking into account terrain properties.
  • Upon reaching the position, the agent is ready to be transported (the report "waiting for carriers" is displayed).

Specific cases

Comments

End of mission

The mission ends once the unit has been unloaded from carrier (even if the transportation is not complete, for instance if the carrier has been destroyed ).
