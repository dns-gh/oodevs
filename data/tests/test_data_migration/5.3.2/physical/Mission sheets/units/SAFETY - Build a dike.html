Mission : SAFETY - Build a dike

Context

To construct a dike to protect sites from floods.
In SWORD, to create and build an object that may prevent areas or sites from floods. As the flood can't cover an area whose altitude is too high, the constructed dike must be built with sufficient height.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Dike location

The work location for the dike.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Behavior

The agent moves toward a construction position to create the dike. The position to create the dike is automatically determinated by the agent (on the edge of the work location).
Upon reaching the construction position, the agent starts to create and construct the dike (the "work just started" report should be displayed). Upon construction termination (the report "work completed for object [object name]" is displayed) the mission ends.

Specific cases

The dike may not be created. Possible causes are:
  • the agent doesn't have the physical capability or it hasn't enough resources.
  • the dike object definition is wrong (the object capacity "construction" for the 'dike' object is not set in the authoring tool).
If this case, the report 'Impossible work' is displayed, and the mission ends.

Comments

If needed, the agent dismounts to build the dike.

End of mission
