Mission : SAFETY - Create and operate a decontamination site

Context

To operate an NBC decontamination site in order to ensure unit decontamination.
In SWORD, operating a decontamination site allows the agent to complete decontamination missions. It decontaminates contaminated units (contaminated units may be issued the "go get decontaminated" mission).

Parameters

Mandatory

Danger direction

Expected enemy direction.

Decontamination site location

The location at which the agent creates the decontamination site.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Remove installation (default value is TRUE)

Determines whether the agent should deconstruct the decontamination site when the mission ends.

Behavior

The agent moves to a position that allows the construction of the decontamination site (a close position on the edge of the planned object) and starts to construct it ("work just started" reports should be displayed). Upon construction termination (the report "work completed for object [object name]" should be displayed), the agent moves toward the center of the decontamination site and operates it ("Element is being operated" report should be displayed).
It remains there until mission termination.

Specific cases

The decontamination site may not be constructed if:
  • the agent does not have the capability to create the object (no physical capability or insufficient resources)
  • the decontamination site object definition is wrong (the object capacity "construction" for the 'decontamination site' object is not set in the authoring tool)
In this case, the mission ends with the report "work impossible, no construction capacity".

Comments

End of mission

The mission continues until another mission.
