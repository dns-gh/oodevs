Mission : SAFETY - Create and operate a checkpoint (filter crowds)

Context

To operate a checkpoint in order to control and filter movements of any kind of entity.
In SWORD, creating and operating a Checkpoint allows the agent to filter crowds. Filtering crowds allows the agent to detect hidden units (terrorist or other characters).

Parameters

Mandatory

Danger direction

Expected enemy direction.

Checkpoint location

The location at which the agent creates the checkpoint.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Intensity (%, default is 100%)

The strength that is used to filter crowds. A value of '0' means "no filtering at all". A value of '100' means "totally blocked" (the crowds that interact with the checkpoint cannot go through it)

Remove installation (default value is TRUE)

Determines whether the agent should deconstruct the checkpoint when the mission ends.

Behavior

The agent moves to a position that allows the construction of the checkpoint (a close position on the edge of the object to create) and starts to construct it ("work just started" reports should be displayed). Upon construction termination (the report "work completed for object [object name]" should be displayed), the agent moves toward the center of the created checkpoint and uses it to filter ("Element is being operated" report should be displayed) the crowd.
It remains there until mission termination.

Specific cases

The checkpoint may not be constructed if:
  • the agent does not have the capability to create the object (no physical capability or insufficient resources),
  • the checkpoint object definition is wrong (the object capacity "construction" for the 'checkpoint' object is not set in the authoring tool).
In this specific case, the mission ends with the report "work impossible, no construction capacity" displayed.

Comments

End of mission
