Mission : SAFETY - Go get decontaminated

Context

Units may be contaminated during an NBC incident. To avoid NBC contamination and intoxication of other units (contamination spreads and contaminates other units), contaminated units should be decontaminated with appropriate means.
In SWORD, an agent must go through a decontamination site object to get decontaminated. The decontamination process is effective if the decontamination site is operated by other units.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Decontamination site

The decontamination site to use to get decontaminated.

Garage position

the position to reach after being decontaminated.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Behavior

The agent moves toward the issued decontamination site. Once there, the agent requests decontamination. The report "request for decontamination" should be displayed. The agents then get decontaminated within a period of time depending on:
  • the number of units that are operating the site (if the site is not operated, the agent stays next to it until the decontamination site is operated or another mission is issued),
  • the number of units asking to get decontaminated on the site (the agent needs to queue to get decontaminated; if the agent asked first, it should be the first to get decontaminated).
Once the agent gets decontaminated (the report "unit decontaminated" should be displayed), it moves toward the garage position and the mission ends.
