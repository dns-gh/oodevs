Mission : IMINT - Operate drones

Context

The systematic observation of aerospace, surface or subsurface areas, places, persons, or things, by visual, aural, electronic, photographic, or other means.
To provide continuous view, and the potential for reports on the activity of an objective.
In this Sword mission, to move to a given position and operate a drone (UAV). Thus, it is usually a mission preliminary to the UAV "IMINT Collect informations" mission.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Position

Position onto which the agent operate the drone (empty by default).

Number of UAV to operate

Number of UAV to operate, 1 by default. It will not operate more drones that it have.

Behavior
  • The agent moves to the issued position. If it is not reachable (a urban block inside which the agent cannot move for instance), the agent moves as close as possible, else it moves to the center of the position. Average speed depends on the physical capabilities regarding to terrain properties.
  • If no position issued, the agent stays on its own position.
  • Upon reaching position, the agent deploys onto it in order to operate as many drones as the optional parameter (by default, it deploys only one) ( the reports "starting deploy" and then "section deployed" should be displayed; for the drone "aircraft is being operated" and" aircraft has been operated" ). The delay of deployment depends on the data defined in authoring tool (see "units" tab, "deployment" section).
  • Note that the agent chooses a drone among one that are available to be operated. The corresponding drones are the one that have not been already operated, or the one that are not already performing a mission. If the agent cannot operate a drones, the report "no drone available" is displayed. In that case the mission terminates.
  • Upon deployment termination, the agent has operated a drone (UAV) that will be ready to be launched. To verify that an UAV agent can be launched, the UAV agent should display "aircraft ready".

Comments

End of mission

Once a drone has been operated, the mission ends. The agent can continue to operate other drones using that mission.
