Mission : SAFETY - Extinguish fires within area

Context

To fix, control and extinguish fires that threaten poulations or structures.
In SWORD, extinguishing fires can prevent alerting or evacuating populations from urban blocks. Only occasional fires are used to render the effect of arsons and local fires, to stimulate the intervention of firefighters. They cannot propagate, but can be extinguished by specific units. When an occasional fire is extinguished, it disappears. A fire decreases depending on the decrease rate. When a fire reaches 0, it is extinguished. Extinguishing a fire with specific units applies additional decrease rates on a fire.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Area

The area inside which the fires need to be extinguished.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Behavior

The agent moves toward each fire (the agent determines an extinguishing position) which is totally inside the specified area (the order of traversal is randomly defined).
Upon reaching a fire, the agent acts to extinguish it. Once the fire is extinguished it moves to the next one.

Specific cases

A fire may not be extinguished. Causes can be:
  • the fire is already extinguished (the fire goes out while the agent moves toward it),
  • the agent doesn't have the physical capability (no dotations or no sutiable extinguisher, for instance).
In these specific cases, the agent displays the report "fire cannot be extinguished" and moves toward the next one.

Comments

The delay of fire extinction depends on:
  • the number of units acting to extinguish the fire and their physical capabilities
  • the weather (rain)
Note that the time taken can be very long if the agent doesn't have sufficient "capability" to extinguish a fire. In this case, reinforcements should be ordered.

End of mission

When there are no more fires to extinguish, the agent stays next to the last visited fire and the mission ends.
If there are no fires within the area, the mission ends and the report "no fire within area" is displayed.
