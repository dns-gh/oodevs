Mission : SAFETY - Resume construction works

Context

To resume construction of something.
In SWORD, to know how to build a set of existing objects. This mission can be used to finish the construction of already started works. It could be used to model the repairing of damaged installations or urban blocks.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Works

the objects (works) to construct,

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Wait for reinforcement (default value is FALSE)

defines if the agent needs to wait for potential reinforcement in case it cannot construct a work.

Instantaneously (default value is FALSE)

define if the agent builds issued elements (objects or urban blocks) instantaneously (magically) or not.

Behavior

The agent moves toward each work to construct. The order of the elements into the 'Works' list defines the order of visit. The positions reached to construct each object are automatically determinated by the agent.
Upon reaching a construction position, the agent starts to construct the object (the "work just started" report should be displayed). Upon construction termination (the report "work completed for object[object name]" should be displayed), the agent moves to the next work to do (next construction position for the next objet to construct).

Specific cases

If the parameter "Instantaneously" is set to TRUE, the agents creates and builds each issued work magically. In that specific case, the object construction do not depends on the agent's physical capabilities or the object definition.
A work may not be constructed. If the agent doesn't have the physical capability or if it hasn't enough resources to do it. In that specific case, the agent behaves in two ways depending on the 'waitForReinforcement" parameter:
  • if it is set to FALSE, the report 'Impossible work' is displayed, and the agent moves to the next work to construct;
  • if it is set to TRUE, the agent will stay nearby the object and ask for reinforcement. It won't move to the next work until it is totally constructed.

Comments

If needed, the agent dismounts to construct an object.

End of mission

The mission is done when all works have been visited (even if some of the objects couldn't be built).
