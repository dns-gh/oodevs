Mission : COMBAT - Escort refugees to camp

Context

A member of the Armed Forces assigned to accompany, assist, or guide an individual or group.
An armed guard that accompanies a convoy, a train, prisoners, etc.
In SWORD, to accompany refugee units toward a specific location (a position or a camp, for instance).

Parameters

Mandatory

Danger direction

Expected enemy direction.

Camp

Camp into which the agent has to accompany the refugee units.

Refugees

Refugee units to be transported to the given camp.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Behavior
  • The agent moves toward each given refugee unit, from the nearest to the furthest.
  • Average speed depends on the physical capabilities regarding the terrain properties.
  • Upon reaching a refugee unit, the agent loads it (magic tranportation).
  • Once all units have been loaded, they are tranported to the given camp.
  • Upon reaching the camp (the agent moves to the center of it), all transported refugees are unloaded. They are taken into account by the logistics unit of the camp.

Specific cases
  • If a refugee unit is not reachable (the unit is located inside an urban block inside which the agent cannot move, for instance), the agent moves as close as possible.
  • If a refugee unit is moving the agent will follow it until it is reached.
  • Note that if the issued object is not a camp (regarding the objects capacities in authoring tool), the mission ends with a failed status (the report "no camp available")

Comments

End of mission

The mission ends once all the refugee units have been transported on the given camp.
