Mission : NBC - Create and operate a decontamination site

Context

To operate an NBC decontamination site in order to ensure unit decontamination.
In SWORD, to build and operate a decontamination site object. Operating a decontamination site allows the agent to complete decontamination missions using that site. It decontaminates contaminated units (contaminated units may be issued the "Get decontaminated" mission).

Parameters

Mandatory

Danger direction

Expected enemy direction.

Planned site

The decontamination site object to be operated.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Behavior
  • The agent moves toward the planned work to build (to build the object, the agent automatically determines a position on the border of the planned object). If the position is not reachable, the agent moves as close as possible.
  • Upon reaching the work position the agent starts to build the object.
  • Once the decontamination site has been built, the agent moves to the center of the object and starts to operate it (the report "Element is being operated" should be displayed).

Specific cases

Comments
  • The decontamination site may not be created if the agent does not have the capacity to build it in terms of physical capability and dotations. In this specific case, the report "work impossible, no capacity" is displayed and the mission ends.
  • The decontamination site may not be operated if it simply cannot be operated (see object capacity "workable" in authoring tool). In this case, the report "element cannot be operated" is displayed and the mission ends.
  • Note that no specific capabilities are required for the agent to operate a decontamination site.
  • The agent cannot be contaminated by other units while operating a decontamination site.

End of mission

This mission continues until another mission is issued.
