Mission : AIR - MoveTo

Context

To fly to an objective in order to reach it.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Route

Objective to reach.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Height

Height to be reached by the agent (in meters).

Behavior

The agent takes off at the specified height and moves toward each waypoint defining the route.

Specific cases

Comments

End of mission

When there are no more waypoints to visit, the agent lands (the agent's altitude is 0 meters) and the mission ends.
