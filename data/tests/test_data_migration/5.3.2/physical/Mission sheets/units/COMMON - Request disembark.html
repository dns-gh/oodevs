Mission : COMMON - Request disembark

Context

To disembark on a position when being transported by other unit(s)
In Sword, to request the disembarking of an agent on an disembarking position. This mission is used to coordinate the troops and vehicules transportation (see transportation capabilities in authoring tool), when SWORD is connected to other simulations through an HLA federation.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Disembarking position

Disembarking time

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Behavior
  • The transporting units move toward the given position. If it is not reachable (a urban block inside which the units cannot move for instance), the units move as close as possible, otherwise they move to the center of the position. Average speed depends on the physical capabilities regarding the terrain properties.
  • Upon reaching the position, the agent is ready to be disembarked.

Specific cases

Comments

End of mission

The mission ends once the agent has disembarked.
