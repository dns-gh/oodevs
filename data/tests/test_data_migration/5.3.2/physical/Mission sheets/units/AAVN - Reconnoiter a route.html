Mission : AAVN - Reconnoiter a route

Context

A mission undertaken to obtain information by visual observation, or other detection methods, about the activities and resources of an enemy, or about the meteorological, hydrographic, or geographic characteristics of a particular area.
In SWORD, to move along a route and to engage all enemies in range.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Route

The set of objectives to reach.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Permanent

Whether or not the mission is permanent. False by default.

Aviation range

The aviation range enforced during the supplying.

Rally point

The rally point to go to when the last position of the route has been reached (if the mission is not permanent) or if the mission ends.

Behavior
  • The agent moves toward the first objective the. If it is not reachable (an urban block inside which the agent cannot move, for instance), the agent moves as close as possible, else it moves to the center of the objective. Average speed depends on the physical capabilities and terrain properties.
  • While moving toward the objective, the agent engages all enemies it encounters.
  • Upon reaching the first objective the agent moves to the next one with the same behavior, until the last one is reached. Then :
    • If the mission parameter "Permanent" is set to false (default value), then :
      • If a rally point is provided, then the agent moves to the rally point and ends its mission.
      • Otherwise, the agent ends its mission.
    • Otherwise, the agent stays at the last waypoint and does not end its mission.

The agent receives a set of points to reconnoiter as parameters. These points usually are keypoints to reconnoiter, such as crossroads, factories, etc.

The agent has detected several enemies, and has engaged some of them.

Specific cases

Comments

If an "End mission" fragmentary order is given, then the agent will end its mission and move to the rally point. If the mission is permanent, then this is the only use of the "rally point" parameter in this mission.

End of mission

If the mission parameter "Permanent" is set to false (default value), then the mission ends when the agent reaches the rally point (or the final waypoint of the route if no rally point was provided). Otherwise, the mission never ends.
