Mission : LOG - Deploy (on a position)

Context

Set-up adopted by a unit for immediate or delayed action.
In SWORD, for logistics agents, to move toward a deployment position and self-deploy to it. A deployed logistics agent enables its logistics chains.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Deployment position

Position to reach to deploy onto it (empty by default).

Behavior
  • If the 'Deployment position' is not issued the agent deploys on its own position.
  • If a position is issued, the agent moves toward it and self-deploys to it (the report "starting to deploy" then section deployed" should be displayed). The deployment may take some time (see "Deployment" section in "Units" tab).
  • Upon deployment termination, the agent activates its logistics chains.

Specific cases

Comments
  • If needed, the agent dismounts.

End of mission

The mission continues until another one is issued.
