Mission : LOG - Sort and heal wounded

Context

In SWORD, for logistics agents, to know how to move toward a deployment position and self-deploy to it. A deployed logistics agent enables its medical chains to sort and heal wounded.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Deployment position

The position to reach.

Wounds priorities

List of the wounds priorities to heal (medical priorities: U1, U2, U3, UE).

Behavior
  • The agent self-deploys onto its own position (the report "starting to deploy" then "section deployed" should be displayed). The deployment may take delays (see "Deployment" section in "Units" tab).
  • Upon deployment termination, the agent the activates its medical chains.
  • If the 'Wounds priorities' parameter is not issued, the agent heals wounds with a priority depending on the severity (UE, U3, U2, U1).
  • If the 'Wounds priorities' parameter is issued, the agent heals wounds with a priority depending on the order of the elements into the 'Wounds priorities' list.

Specific cases

A deployment position is provided:
  • The agent moves to the position. If the objective is not reachable, the agent moves as close as possible (the agent can move within it), the agent moves to the center, or to the closest position to the center). Average speed depends on physical capabilities, taking into accout terrain properties.
  • Upon movement termination, the agent self deploys and activates its medical chains.

Comments
  • If needed, the agent dismounts.

End of mission

The mission continues until another one is issued.
