Mission : COMMON - Disembark

Context

To disembark units on a position when transporting them.
In Sword, to disembark units in an area, regardless of the physical capacities of the carrier (magic transportation).

Parameters

Mandatory

Danger direction

Expected enemy direction.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Units to disembark

The units to disembark (units already being transported by the agent).

Disembarking position

The position where the disembarking will take place.

Behavior
  • The agent moves toward the given disembarking position (if no disembarking position is provided, then the agent will stay put). If it is not reachable (a urban block inside which the agent cannot move for instance), the agent moves as close as possible, else it moves to the center of the position. Average speed depends on the physical capabilities regarding the terrain properties.
  • Upon reaching the position :
    • If the units to disembark have been manually provided, then only these units will be disembarked.
    • Otherwise, all the currently transported units will be disembarked.

Specific cases

Comments
The agent moves to the disembarking position.

At the end of the mission, the unit has been disembarked by the agent (here, the agent has been given a *Move to* mission right after the end of the *Disembark* mission to show that the transported unit has been disembarked)."

End of mission

The mission ends once all the units that were supposed to be disembarked are disembarked.
