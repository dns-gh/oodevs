Mission : HUMINT - Collect information (in area)

Context

The systematic observation of aerospace, surface or subsurface areas, places, persons, or things, by visual, aural, electronic, photographic, or other means. To provide continuous view, and the potential for reports on the activity of an objective
In this Sword mission, to move to a position and get information about the presence of units within it. It is used to simulate the capability of human intelligence units to gather information, by applying a magic effect in the area. This effects allows the agent to detect (quickly) other units in the area.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Area

The area from which to retrieve information.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

the right or left boundaries inside which the agent perform the mission.

Limit 2

the right or left boundaries inside which the agent perform the mission.

Infiltration mode

Boolean, defaut value is FALSE. Defines if the agent uses infiltration to reach the barycenter of the area.

Position

(location composite, defeult is NIL) The position from which the agent detect units within the area.

Behavior
  • The agent moves toward the position. If no position is issued, the agent moves directly within the area (it moves to the center of the area if it is reachable, otherwise it moves to a close position (the closest position to the center)).
  • Upon reaching the area, the agent starts to detect units by magically reconnoitering the area (the report "reconnaissance started" is displayed, meaning that the agent is able to detect units in the area).
  • If the 'Infiltration mode' parameter is set to 'true', the agent becomes stealthy to simulate the fact that it is able to infiltrate through the enemies positions and hide in the area. Agents of the simulation won't be able to detect it unless the position is searched by specific units (with the 'Reconnoiter' mission for instance). When this parameter is set to 'true', a specific report is displayed "infiltrating" meaning that the agent is no longer visible to other agents.

Specific cases

Comments

Note that the agent is no longer invisible if the mission terminates with, for instance, an 'End mission' fragmentary order.

End of mission

The mission continues until another one is issued.
