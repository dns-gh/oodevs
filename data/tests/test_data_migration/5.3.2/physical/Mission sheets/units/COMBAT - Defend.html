Mission : COMBAT - Defend

Context

To hold a defined object against an enemy attack to halt or ward off an attack in order to defeat or destroy the enemy.
In Sword, to move and occupy an objective (or a set of objectives) and engage approaching enemies.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Objectives

Objective(s) to defend.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Obstacles

Planned obstacles to errect.

Behavior
  • The agent chooses and moves toward the nearest objective. If one of the objectives is not reachable (an urban block inside which the agent cannot move, for instance), the agent moves as close as possible, otherwise it moves to the center of the position. Average speed depends on the physical capabilities regarding the terrain properties.
  • Upon reaching the objective, the agent starts to observe and wait for enemies to come.
  • If enemies are detected, the agent engages them from of the objective. Note that the agent may change its position while engaging the enemies.
  • The choice of the position is made among the objectives. It won't leave the objectives.

Specific cases

Comments

End of mission

The mission ends if another mission (or "the cancel maneuver" fragmantary order) is issued by a user.
