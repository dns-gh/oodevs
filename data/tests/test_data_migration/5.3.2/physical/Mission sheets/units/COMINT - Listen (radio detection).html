Mission : COMINT - Listen (radio detection)

Context

The systematic observation of aerospace, surface or subsurface areas, places, persons, or things, by visual, aural, electronic, photographic, or other means.
To provide continuous view, and the potential for reports on the activity of an objective.
In this mission, to observe an area with radar in order to detect movement of unknown enemies. The agent simply activates its radar sensors within the given area.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Area

Objective in which the agent listens to radio emissions.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Position

Position from which the agent observes the objective (default: empty).

Behavior
  • The agent moves toward the given position. If the position is not reachable (an urban block inside which the agent cannot move, for instance), the agent moves as close as possible, else it moves to the center of the position. Average speed depends on the physical capabilities, taking into account terrain properties.
  • If not issued, the agent stays at its own position.
  • Upon reaching the position, the agent deploys to it. The time required for deployment depends on the data defined in the authoring tool (see "Units" tab, "Deployment" section). Upon deployment termination the report "section deployed" should be displayed.
  • Upon deployment termination, the agent activates its radars and starts to monitor the area (the reports "radar silence lifted" and "monitoring: started" are displayed). Unknown units located in the area with radio emission switched to "on" should be detected.

Comments

End of mission

The mission ends if another mission (or "the cancel maneuver" fragmentary order) is issued by user.
