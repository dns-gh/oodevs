Mission : COMBAT - Harass

Context

Mission to destroy or at least to neutralize enemy hidden elements, infiltrated or during infiltration and exfiltration in a given area.

Parameters

Mandatory

Danger direction

Objective

Position where you want to harass enemies.

Optional

Phase lines

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Entities to harass

Enemies agents to harass.

Destroying positions

Positions for observing objective where begin to harass enemies.

Obstacles

Obstacles to build.

Behavior

The agent reaches a position (selected automatically by the agent) to observe the objective. The report 'zone control: started' should be displayed. The agent will therefore observe this objective to be able to engage the identified enemy elements. Any identified enemy that passes inside the objective (area or urban block) will, if possible, be destroyed. If some destroying positions are given, the agent moves between them. When the agent stops to observe the objective, the report 'zone control: stopped' should be displayed.

Specific cases
  • destroying positions:
    • If no destroying positions are given:
      • the agent reaches the center of the objective if possible;
      • otherwise, the agent reaches the nearest postion that it can reach in the objective.
    • If at least one destroying position is given as parameter:
      • the agent reaches one of the destroying positions and observes the objective. The choice of the destroying position is automatically computed by the agent.
      • If an enemy could be engaged, the agent could move from the destroying position to other destroying positions in order to find the best position to apply direct fire and engage the enemy.
  • If obstacles are given as parameters:
    • the agent will construct them at the beginning of the mission if the agent has the capacity and the dotation to build them.
    • if withImprovement parameter is true, the obstacles will be improved (mining) by the agent if the agent has the capacity and the dotation to mine it.
  • If enemy agents are given as parameters, engage them, otherwise engage any enemy located inside the objective.

Comments

End of mission

The mission continues until another mission is issued.
