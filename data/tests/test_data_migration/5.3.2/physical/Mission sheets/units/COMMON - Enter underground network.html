Mission : COMMON - Enter underground network

Context

To get into an underground network such as sewer, metro...
In Sword, to move toward a network entrance (a sowrd knowledge object) in order to get in underground network. This mission is priliminary to the mission "move within an underground network.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Entrance

Entrance to get in the underground network.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Behavior
  • The agent moves toward the given underground entrance. If it is not reachable (the object is located inside a urban block which the agent cannot move within), the agent moves as close as possible. Average speed depends on the physical capabilities regarding the terrain properties.
  • Upon reaching the underground entrance, the agent dismounts (if needed,) and enters the network.

Specific cases

While moving through an underground network entrance, the agent won't be able to ENTER if:
  • the underground entrance is deactivated. In this case the agent waits for the entrance to be activated. Once it is done (with a magic frag order on the object, for instance) the agents gets out and finishes the mission in a normal way.
  • the underground entrance is destroyed (the object disappears): in this case the agent moves to the entrance position and the mission has failed.

Comments

End of mission

The mission ends once the unit is in the underground entrance.
