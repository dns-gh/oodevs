Mission : OOTW - Operate a checkpoint (filtrate crowds)

Context

To operate a checkpoint in order to filter crowds and detect infiltrating terrorists.
In SWORD, operating a checkpoint (a Sword object can have the ability to filter crowds) allows the agent to detect terrorists, disarm and control movements of encountered crowds.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Checkpoint position

The location at which the agent will create the checkpoint.

Safety position

The position to reach when the agent cannot stand in front of uncontrolled crowds.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Body search strength

The intensity of the body search at the checkpoint.
A value of 100% means that all armed persons in the crowd will be disarmed. The stronger this is, the harder it is for the crowd to cross the checkpoint.

Behavior
  • The agent moves to the work location and creates (magically) a checkpoint (the report "control point established" should be displayed).
  • Then the agent moves to the center of the checkpoint in order to operate it (the report "element [object name] is being operated should be displayed).

Specific cases
  • If the work location is not reachable, the agent moves as close as possible, otherwise it moves to a close position (the position closest to the center of the future object) and the agent builds the object. In this case, it is possible that the object is not built at the exact location.
  • If encountered crowds cannot be controlled (see specific reports) the checkpoint may be destroyed and the agent may protect itself by moving to the given safety position.

Comments

This mission has a few different effects:
  • encountered crowds are disarmed with a ratio depending on the given 'body search strength' parameter
  • encountered crowds are controlled by the agent. If not, the agent protects itself and the checkpoint is destroyed.
  • the filtration intensity depends on the ratio of persons to disarm (if the "body search strength" is 0, the filtration intensity will be low, if the "body search strength" is high, the filtration will be high).
  • if terrorists hide in an encountered crowd, the agent may capture them.
Note that terrrorists are captured only if they cross the effective area of the checkpoint. They cannot be detected with normal sensors. Here only the checkpoint (see 'Objects' capacity in the authoring tool) is able to ensure the detection and capture of a terrorist by the agent.
