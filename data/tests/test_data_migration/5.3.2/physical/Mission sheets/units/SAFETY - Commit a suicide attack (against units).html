Mission : SAFETY - Commit a suicide attack (against units)

Context

Terrorists may perform suicide attacks intending to kill and maim innocent people.
In SWORD, to reach a specific unit and launch a bomb projectile on its own position, causing the death of the terrorist and injuring the target unit.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Target unit

the agent to attack

Resource

type of "bomb" to use (type of ammuntion to use)

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Behavior

The agent follows the target unit until its position is reached. It then launches the chosen "bomb" projectile on its own position and dies.

Specific cases

Because the mission uses the indirect fire capability of the agent, it is possible that the agent willl fail to launch the projectile on the target unit. Causes could be:
  • the agent doesn't have the physical capability (no suitable launcher with regard to dotation type): the report "No launcher with capacity to use specified munition" is displayed.
  • the agent does not have enough dotation: the report "no more shells" is displayed.
  • the selected resource type cannot allow the agent to perform indirect fire: the report "Impossible indirect fire" is displayed.

Comments

End of mission
