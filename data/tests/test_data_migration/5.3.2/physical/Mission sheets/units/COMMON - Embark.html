Mission : COMMON - Embark

Context

To embark units on a position to transport them.
In Sword, to embark units in an area, regardless of the physical capacities of the carrier (magic transportation).

Parameters

Mandatory

Danger direction

Expected enemy direction.

Embarking area

The area within which the units will be embarked.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Units to embark

The units to embark.

Interaction distance

The maximum distance (in meters) between the agent and the unit to load (default value is 100 meters).

Behavior
  • The agent moves toward the given embarking area. If it is not reachable (a urban block inside which the agent cannot move for instance), the agent moves as close as possible, else it moves to the center of the position. Average speed depends on the physical capabilities regarding the terrain properties.
  • Upon reaching the position :
    • If the units to embark have been manually provided, then only these units will be embarked (if they are in the embarking area, and if the distance between the units to embark and the agent is lesser than the "Interaction distance" parameter).
    • Otherwise, all the friendly units in the area will be embarked.

Specific cases

Comments
The agent moves to the embarking area to embark the unit.

At the end of the mission, the unit has been embarked by the agent."

End of mission

The mission ends once all the units that were supposed to be embarked are embarked.
