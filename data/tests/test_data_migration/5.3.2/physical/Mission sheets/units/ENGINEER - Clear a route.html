Mission : ENGINEER - Clear a route

Context

To reduce or negate the effects of existing or reinforcing obstacles. The objective is to maintain freedom of movement for maneuver units, weapons systems and critical supplies.
In Sword, to move along a computed path and remove obstacles that obstruct the path. Here, 'obstacles' are Sword 'objects' that have the capacity "avoidable" and/or "constructor".

Parameters

Mandatory

Danger direction

Expected enemy direction.

Waypoints

Waypoints defining the route to follow (the path will be computed taking into account those waypoints).

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Behavior
  • The agent moves along the computed path. If a non-bypassed 'obstacle' obstructs the path, the agent tries to reaches it.
  • Upon reaching an obstacle, the tries to remove the obstacle. Before removing the obstacle, if it is improved with mines, the agent tries to remove the improvement. If it fails (the agent has not the capacity to do it), the report "the agent cannot remove mines from the object" should be displayed. If the agent has the capacity, it starts to remove mines from it (the report "demining the object [object name]" should be displayed). Then, two possibilities:
    • the obstacle can be removed: the agent starts to remove it (the report "work just started" should be displayed).
    • the obstacle cannot be removed (if the agent doesn't have the physical capability or it hasn't enough resources), the report 'Impossible work' is displayed. In this case the agent waits for reinforcement.
  • Upon work termination (the report "work completed for object [object name]" is displayed) and the agent continues to move along the path.
  • If required, the agent dismounts to perform the work.

Specific cases
  • Note that the delay can be very long (it can depend on the type and the shape of the obstacle).

Comments

End of mission

When the last waypoint has been reached, the mission terminates.
