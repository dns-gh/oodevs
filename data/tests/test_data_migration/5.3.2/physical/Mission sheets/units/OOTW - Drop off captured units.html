Mission : OOTW - Drop off captured units

Context

A mission to drop off captured units in a position.
In Sword, to reach a position and to drop off the required units. Units may be transported by the agent.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Position

The location at which the agent has to unload the given units.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Captured units

The units to be unloaded at the given location. If parameter is not filled, the agent will drop off all captured units.

Behavior
  • The agent moves to the given position. If the position is not reachable (an urban block the agent enter, for instance), the agent moves as close as possible, otherwise it moves to the center of the position. Average speed depends on the physical capabilities with regard to terrain properties.
  • The agent moves to the position given by user.
  • Then, the agent unloads the "captured units" given by user otherwise the agent unloads all captured units.

Specific cases

The selected units are not being transported by the agent. In that case, the agent emits the report "the selected unit is already unloaded" should be displayed.

Comments

This mission is not intended to liberate prisoners : if the provided units are prisoners, they will be dropped off as usual, but they still won't be able to communicate.

End of mission

The mission ends when all captured units have been unloaded at the position.
