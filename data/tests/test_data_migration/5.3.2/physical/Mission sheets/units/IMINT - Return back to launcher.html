Mission : IMINT - Return back to launcher

Context

Fly back to the launcher unit.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Behavior
  • The agent takes off, flies back to the drone station if it has not moved too far away (if another mission has been issued for instance) and lands. Once the agent has been retrieved (it is loaded onto the drone station) it can be operated again.

Specific cases
  • If the agent is landed, it takes off alone (without its CP unit) and flies back to the CP.

Comments

End of mission
