Mission : COMBAT - Provide indirect fires (from a position)

Context

The action of a force which aids, protects, complements, or sustains another force in accordance with a directive requiring such action.
  • A unit which helps another unit in battle. Aviation, artillery, or naval gunfire may be used as a support for infantry.
  • A part of any unit held back at the beginning of an attack as a reserve.
  • An element of a command which assists, protects, or supplies other forces in combat.
In this mission, to provide INDIRECT FIRES to a set of units, in order to destroy or neutralize dangerous enemies.

Parameters

Mandatory

Danger direction

Position

Supporting position from which the agent provides indirect fire.

Friends

Friendly units to support.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Behavior
  • The agent moves toward the given position. If it is not reachable (a urban block inside which the agent cannot move for instance), the agent moves as close as possible, else it moves to the center of the position. Average speed depends on the physical capabilities regarding the terrain properties.
  • Upon reaching the position, the agent starts provide indirect fires if:
    • a specific fragmentary order is given (FIRE)
    • if the supported units encounter enemies around.

Specific cases

The agent may failed to deliver fires if:
  • the "Fire" order is given onto an objective to far from the the agent (regarding its physical capabilities)
  • the suported units are too far from the agent and enemies around are also (for the same reasons).
  • the agent doesn't have enough ammo.
In that case a specific report is displayed.

Comments

End of mission

The mission ends if another mission is issued.
