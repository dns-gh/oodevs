Mission : ASY - Assassinate officers

Context

A surprise attack involving the deliberate use of violence against civilians in the hope of attaining political or religious aims.
In this Sword mission, the agent moves to the target unit and acts to kill its officer components.

Parameters

Mandatory

Danger direction

Expected ennemy direction.

Target unit

Objective to attack.

Meeting position

Meeting point to reach upon assassination completion.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Behavior

The agent moves toward the issued target. If it is not reachable (an urban block inside which the agent cannot move, for instance), the agent moves as close as possible, else it moves directly to the position. Average speed depends on physical capabilities with respect to terrain properties.
Upon reaching the target, the agent acts to kill the officer components of the unit (the report "attack is being executed" is displayed).
Then, it moves to the meeting position and the missions ends.
Note that the mission functions in the same way if the target unit has no officiers.
If this mission is carried out by a "terrorist" agent, it may not be possible to detect it unless the target unit is defended by safety devices such as check points.
