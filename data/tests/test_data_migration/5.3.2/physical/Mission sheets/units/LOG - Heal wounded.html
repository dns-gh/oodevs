Mission : LOG - Heal wounded

Context

Activating a medical chain for treating wounded.
In SWORD, for logistics agents, to move toward a deployment position and self-deploy to it. A deployed logistics agent enables its medical chains to treat wounded.

Parameters

Mandatory

Danger direction

Optional

Phase lines

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Wounds healed

List of the wounded severity to treat given by operator (medical priorities : U1, U2, U3, UE).

Deployment position

Position to reach to deploy onto it (empty by default).

Behavior
  • If the Deployment position is not issued the agent deploys to its own position.
  • If a position is issued, the agent moves toward it and self-deploys to it (the report "starting to deploy" then "section deployed" should be displayed). The deployment may take time (see "Deployment" section in "Units" tab).
  • If Wounds healed is not issued the agent treats all kind of injuries (UE, U3, U2, U1).
  • If Wounds healed is issued the agent heals the kind of injuries given by the operator.
  • Upon deployment termination, the agent the activates its medical chains.

Specific cases

Comments
  • If needed, the agent dismounts.

End of mission

The mission continues until another one is issued.
