Mission : SAFETY - Build a firebreak

Context

To construct a firebreak to protect sites from fires.
In SWORD, to know how to create and build a firebreak object that may prevent areas or sites from spreading fires.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Firebreak location

The work location for the firebreak.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Behavior

The agent moves toward a construction position to create the firebreak. The position to create the firebreak is automatically determined by the agent (on the edge of the work location).
Upon reaching the construction position, the agent starts to create and construct the firebreak (the report "work just started" should be displayed). Upon construction termination (the report "work completed for object [object name]" is displayed) the mission ends.

Specific cases

The firebreak may not be created. Possible causes are:
  • the agent doesn't have the physical capability or it hasn't enough resources.
  • the firebreak object definition is wrong (the object capacity "construction" for the 'firebreak' object is not set in the authoring tool).
If this case, the report 'Impossible work' is displayed, and the mission ends.

Comments

If needed, the agent dismounts to build the firebreak.

End of mission
