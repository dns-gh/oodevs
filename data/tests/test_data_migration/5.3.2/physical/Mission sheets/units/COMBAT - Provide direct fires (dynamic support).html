Mission : COMBAT - Provide direct fires (dynamic support)

Context

The action of a force which aids, protects, complements, or sustains another force in accordance with a directive requiring such action.
  • A unit which helps another unit in battle. Aviation, artillery, or naval gunfire may be used as support for infantry.
  • A part of any unit held back at the beginning of an attack as a reserve.
  • An element of a command which assists, protects, or supplies other forces in combat.
In SWORD, supporting a unit is to provide intelligence and DIRECT FIRES, in order to destroy or neutralize dangerous enemies, by moving from one supporting position to another.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Units

Friendly units to support.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Retrograde context

Defines if the mission is performed in a retrograde context (e.g 'screen' or 'delay' company missions)(default value is 'false').

Behavior
  • The agent chooses and moves toward a supporting position to get ready to provide fire to supported units. If the chosen position is not reachable (an urban block inside which the agent cannot move, for instance), the agent moves as close as possible, else it moves to the center of the position. Average speed depends on the physical capabilities while taking into accout terrain properties.
  • The chosen position can change dynamically if the supported agents moves. In that case, the agent moves from one position to another.
  • Upon reaching a supporting position, the agent fires at enemies that engage supported friends.

End of mission
