Mission : ENGINEER - Disable a resource network

Context

To disable a resource network node.
In SWORD, to disable a resource network node.

Parameters

Mandatory

Danger direction

Resource network

Resource network node to disable (several kinds of resource network could be located in the same location: only one of them can be disabled during the mission).

Meeting position

Position to reach after resource network node has been disabled.

Optional

Phase lines

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Behavior
  • The agent moves toward the resource network node to disable. The position to disable the resource network node is automatically determined by the agent.
  • If needed, the agent dismounts at reach the resource network node.
  • The agent tries to enter the urban block if the agent cannot reach the resource network node beacause of the trafficability of the urban block. The agent moves toward the border of the urban block.
  • Upon reaching the resource network node or the border of the urban block, the agent disables the resource network node (the report "Resource node disabled" should be displayed)
  • When the resource network node has been disabled, the agent moves to the meeting position.

Specific cases

Comments

End of mission

The meeting position is reached.
