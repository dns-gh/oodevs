Mission : NBC - Operate a decontamination site

Context

To operate an NBC decontamination site in order to ensure unit decontamination.
In SWORD, operating a decontamination site allows the agent to complete decontamination missions using that site. It decontaminates contaminated units (contaminated units may be issued the "Get decontaminated" mission).

Parameters

Mandatory

Danger direction

Expected enemy direction.

Decontamination site

The decontamination site object to be operated.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Behavior
  • The agent moves to the center of the object and starts to operate it (the report "Element is being operated" should be displayed).

Specific cases

Comments
  • The decontamination site may not be operated if it simply cannot be operated (see object capacity "workable" in the authoring tool). In this case, the report "element cannot be operated" is displayed and the mission ends.
  • Note that no specific capabilities are required for the agent to operate a decontamination site.
  • The agent cannot be contaminated by other units while operating a decontamination site.

End of mission

This mission continues until another mission is issued.
