Mission : COMBAT - Search and destroy

Context

An offensive action characterized by movement supported by fire.
In SWORD, to reconnoiter key points in the area in order to find enemies. All enemies in range will be engaged.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Area

The area within which the agent searches for enemies.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Behavior

* The agent moves to each keypoint inside the given aera. If an element is not reachable (an urban block inside which the agent cannot move, for instance), the agent moves as close as possible. Average speed depends on the physical capabilities while taking into account terrain properties.
* While moving inside the area, the agent engages all enemies in range.

Specific cases
  • Only enemies located inside the area are engaged.
  • If no position is found in the area the agent reconnoiters the area.
  • The agent may move toward an encountered enemy to destroy, when it is located in the area.

Comments

End of mission

The mission continues until another one is issued.
