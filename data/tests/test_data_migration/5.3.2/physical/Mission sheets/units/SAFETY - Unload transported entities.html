Mission : SAFETY - Unload transported entities

Context

In SWORD, unloading a crowd or units allows to empty the transportation vehicles of an agent so that agent can be used at its full capacity for carrying out another transportation mission. If a transportation mission was in progress, it is canceled when the "Unload transported crowd" mission is given.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Target destination

the location to unload transported element

Behavior

The agent moves toward the destination if specified. Once the destination is reached, the agent unloads all transported elements. If the destination is not specified, the agent unloads transported elements on its positions.

Specific cases

If the agent is not transporting any elemement when the mission is given, the agent reports "transport: no crowd to unload". In this case, the agent won't go to the destination if it has been specified.

Comments

End of mission

The mission ends whentransported element by the agent have been unloaded.
