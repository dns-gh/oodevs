Mission : SAFETY - Attack urban blocks within area (with projectiles)

Context

To perform a terrorist attack, exactions against installations and population.
In SWORD, units and crowds can assault urban blocks (or other units and structures). Assaulting an urban block causes damage to its structure and the population within. Attacking urban blocks can be used to highlight a failure in a safety device that should have been set up.
In this mission, the agent damages urban blocks with projectiles (indirect fire).

Parameters

Mandatory

Danger direction

Expected enemy direction.

Area

The area inside which urban blocks must be attacked.

Resource type

The type of resource to use.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Quantity

The number of projectiles to use for each shot.

Behavior

The agent tries to launch the specified quantity of projectiles on each urban block within the area, one by one. It moves toward each urban block which is totally inside the specified area (the order of traversal is randomly defined) until it is at a good distance to launch the projectile (the firing distance is defined by the distance between the agent's position and the center of each target urban block). Once the agent is able to launch the projectile on the current visited urban block, it launches the projectile on it, and will be neutralized for a short time (hiding from the explosion). Once the agent has recovered, it decides to act on the next one.

Specific cases

It is possible that the agent cannot launch the projectile on a target urban block.The causes could be:
  • the agent doesn't have the physical capability (no suitable launcher with regard to dotation type): the report "No launcher with capacity to use specified munition" is displayed.
  • the agent does not have enough dotation: the report "no more shells" is displayed.
  • the selected resource type cannot allow the agent to perform indirect fire: the report "Impossible indirect fire" is displayed.

Comments

End of mission

The mission ends when the agent has tried to launch projectiles on each urban block.
If there are no urban blocks within the area, the mission ends and the report "no urban blocks within area" is displayed.
