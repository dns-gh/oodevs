Mission : OOTW - Capture units

Context

In Sword, to follow a set of entities and capture them.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Units

The units to capture.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Behavior
  • The agent follows the set of entities to capture. If the position of a unit is not reachable (the unit is inside an urban block the agent cannot enter, for instance), the agent moves as close as possible, and waits for the unit to be reachable. Average speed depends on the physical capabilities with regard to terrain properties.
  • Upon reaching one, the agent captures it.
  • The agent continues to the next one until no more units can be captured.

Specific cases

Comments

End of mission

The mission terminates when all units have been captured.
