Mission : ASY - Terrorist attack

Context

A surprise attack involving the deliberate use of violence against civilians in the hope of attaining political or religious aims.
In this Sword mission, the agent moves to the target objective and engages it with indirect fire to simulate a terrorist attack. The agent moves to the target until it reaches its position, launches the projectile onto its own position and dies.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Target

Objective to attack.

Dotation

Dotation type to use (the type of "bomb" to activate).

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Behavior
  • The agent moves toward the objective. If it is not reachable (an urban block inside which the agent cannot move for instance), the agent moves as close as possible, else it moves to the center of the objective. Average speed depends on the physical capabilities with respect to terrain properties.
  • Upon reaching the target, the agent starts to activate its bomb (the report "attack is being executed" is displayed). Because the bomb is simulated with an indirect fire, the explosion can be delayed regarding the bomb activation. This delay depends on the reload time of the weapon system used (see authoring tool, weapon system tab). Because the explosion results from indirect fire, the reports "under artillery fire" and "friendly indirect fire" should be displayed too.
  • Upon bomb activation, all units around should be affected by the bomb, and the agent dies. This terminates the mission.
  • The attack may failed if the given dotation type (the bomb to activate) cannot be used with a short distance target. In that case the report "Attentat impossible" is displayed.
  • If this mission is carried out by a "terrorist" agent, it may not be possible to detect it unless the target unit is defended by safety devices such as check points.
