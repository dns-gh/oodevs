Mission : SAFETY - Increase resource node production

Context

To provide basic needs and resources for populations, where production is insufficient to give an entire population a comfortable standard of living.
In SWORD, to connect to an existing production node and provide a certain quantity of corresponding resources. The node increases its production with the specified quantity.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Resource node

The resource node whose production the user wishes to increase.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Quantity (default value is 100)

The quantity to add to the production of the resource node.

Behavior

The agent moves toward the given resource node (a resource node is located within its corresponding urban block). If the urban block containing the resource node is reachable (the agent can enter it), the agent moves to the center, otherwise it moves to a close position (the position closest to the center).
Upon reaching the resource node, the agent starts to produce in the node. The production of the node increases by the issued quantity.

Specific cases

Comments

End of mission

The production is active until another mission is issued to the agent.
