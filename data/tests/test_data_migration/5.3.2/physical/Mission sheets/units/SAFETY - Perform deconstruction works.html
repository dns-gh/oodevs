Mission : SAFETY - Perform deconstruction works

Context

To deconstruct and remove works or installations.
In SWORD, elements to remove a set of objects. They can be: building collapses, human constructions, mines etc. Removing elements is often required to restore access. When an object is totally decontructed, it disappears.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Works

the objects (works) to deconstruct,

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Wait for reinforcement (default value is FALSE)

defines if the agent needs to wait for potential reinforcement in case it cannot do a work (deconstruct an object).

Instantaneously (default value is FALSE)

defines if the agent can remove objects instantaneously (magically) without any physical capabilities consideration.

Behavior

The agent moves toward each element to deconstruct (the order of traversal is defined by user). The positions reached to deconstruct each object are automatically determinated by the agent.
Upon reaching a deconstruction position, the agent starts to deconstruct the object (the report "work started for object[object name]" should be displayed). Upon deconstruction termination (the report "work destruction completed" should be displayed), the agent moves to the next element to deconstruct.

Specific cases

An object may not be deconstructed if the agent doesn't have the physical capability or if it hasn't enough resources to do it.
If that specific case, the agent behaves in two ways depending on the 'waitForReinforcement" parameter:
  • if it is set to FALSE, the report 'Impossible work' is displayed, and the agent moves to the next work to deconstruct;
  • if it is set to TRUE, agent will stay nearby the object and ask for reinforcement. It won't move to the next object until it is totally deconstructed.
If "instantaneously" parameter is set to 'true', each visited object are instantaneously and magically deconstructed without any physical capability consideration. If this parameter is set to 'true', any object of the world can be removed.

Comments

If needed, the agent dismounts to deconstruct an object.

End of mission

The mission is done when all works have been visited (even if some of the objects couldn't be removed).
