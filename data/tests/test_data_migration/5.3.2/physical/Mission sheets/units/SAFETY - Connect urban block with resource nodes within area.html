Mission : SAFETY - Connect urban block with resource nodes within area

Context

In SWORD, connecting urban blocks with resources nodes allows an agent to create links from a distributing urban block to one or more receiving urban blocks for a given resource.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Urban block (producer)

The producer of the resource network to establish.

Area

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Behavior

The agent moves toward each receivers within the area specified by the operator. When the agent reaches a receiver, it establishes a resource network link from the producer to this receiver.

Specific cases

If one of the receiving resource nodes is on a urban block whose trafficability is too low for the agent, it will report "Cannot reach objective" and skip the connection for this resource node.

Comments

End of mission

The mission ends when all of the receivers have been connected to the producer.
