Mission : COMMON - Offer transport

Context

To move to a position in order to transport other unit(s)
In Sword, to offer the transporting of other unit(s). This mission is used to coordinate the troops and vehicules transportation (see transportation capabilities in authoring tool), when SWORD is connected to other simulations through an HLA federation.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Units to transport

Route

The route through which the agent must transport the crowd, and at the end of which the unloading will take place.

Embarking area

With equipments

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Request identifier

Is partial offer

Behavior
  • The agent moves toward the given position. If it is not reachable (a urban block inside which the agent cannot move for instance), the agent moves as close as possible, else it moves to the center of the position. Average speed depends on the physical capabilities regarding the terrain properties.
  • Upon reaching the position, the agent is ready to transport.

Specific cases

Comments

End of mission

The mission ends once the agent has transported the units.
