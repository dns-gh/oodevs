Mission : SAFETY - Follow

Context

To follow a set of mobile agents (unit or crowd).

Parameters

Mandatory

Danger direction

Expected enemy direction.

Units

the set of entities to follow,

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Positions

the set of positions from which the agent stays as close as possible from entities.

Minimum distance

The minimum distance at which the agent must stay from the entity to follow.

Behavior

The agent will follow the nearest target, trying to find a position to stay as close as possible to the followed entity.
If the positions are issued, agents will choose the position as close as possible to the chosen target (the nearest).
The mission continues until another one is issued.

Specific cases

Comments

End of mission
