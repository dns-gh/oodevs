Mission : SAFETY - Move to

Context

To move to an objective using a defined route.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Waypoints

The waypoints to move along.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Urgency

Behavior
  • The agent moves toward each waypoint defining the route. If a visited waypoint is not reachable (an urban block inside which the agent cannot move, for instance), the agent moves as close as possible, otherwise it moves to the center of the position.
  • Average speed depends on the physical capabilities taking into account terrain properties.
  • Upon reaching a waypoint, the agent moves to the next one and so on, until the final waypoint is reached.

Specific cases

A flying agent receiving this mission will firstly take off, then move toward each waypoint (just like a regular agent), then will land upon arrival to the final objective.

Comments

End of mission

When there are no more waypoints to visit, the agent stays next to the last one and the mission ends.
