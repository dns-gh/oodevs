Mission : COMBAT - Cover (by fire)

Context

Actions to conceal actual friendly intentions, capabilities, operations, and other activities by providing a plausible yet erroneous explanation of the observable.
In Sword, to cover a set of friendly entities by fire. The agent observes the given threat and awaits approaching enemies.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Threat

Objective to monitor.

Positions

Positions from which the subordinates observe the given objective.

Entities

Entities to follow.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Behavior
  • The agent chooses the best position to monitor the given threat. If one of the positions is not reachable (an urban block inside which the agent cannot move, for instance), the agent moves as close as possible, otherwise it moves to the center of the position. Average speed depends on the physical capabilities regarding the terrain properties.
  • Upon reaching a position, the agent starts to observe the threat and wait for enemies to come. All enemies in range will be engaged.
  • If enemies are detected, the agent engages them from the position. Note that the agent may change its position while engaging the enemies.

End of mission
