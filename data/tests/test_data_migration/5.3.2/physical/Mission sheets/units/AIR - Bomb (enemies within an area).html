Mission : AIR - Bomb (enemies within an area)

Context

For an aircraft to fly over targets within area in order to destroy or neutralize them.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Route

The path to move along.

Area

Area inside which targets must be found.

Ammunition

Type of ammunition to use.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Quantity

Quantity of projectiles to use for each shot (default value is 100).

Flying altitude

Height to reach while flying along the route (in meters, default value is 2000 meters).

Behavior
  • The agent takes off at the specified height and moves toward each waypoints defining the route.
  • Upon moving nearby the area, the agent tries to launch the specified quantity of projectile on each encountered enemy found within it.
  • Upon executing indirect fire on a target, the report "Indirect fire completed on target [target name]" is displayed.

Specific cases

A target won't be shot at if:
  • it is not at a good firing distance regarding the selected ammunition: the message "target out of range" is displayed.
  • it is not within the specified area.
  • the agent doesn't have the physical capability (no suitable launcher for the dotation type) : the message "no Launcher" is displayed.
  • the agent does not have enough dotation: the message "no ammo" is displayed.

Comments

End of mission

When there are no more waypoints to visit, the agent reaches its command post, lands (the agent's altitude is 0 meters) and the mission ends.
