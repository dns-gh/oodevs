Mission : SAFETY - Block a road

Context

To construct a roadblock to prevent agents through a road.
In SWORD, to know how to create and build a roadblock object that may prevent movement from agent through this.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Roadblock location

The location at which the agent creates the roadblock.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Remove installation (default value is TRUE)

Determines whether the agent should deconstruct the roadblock site when the mission ends.

Behavior

The agent moves toward a construction position to create the roadblock. The position to create the roadblock is automatically determined by the agent (on the edge of the work location).
Upon reaching the construction position, the agent starts to create and construct the roadblock (the report "work just started" should be displayed). Upon construction termination (the report "work completed for object [object name]" is displayed).
The roadblock is removed at the end of mission by default if optional parameter is set to 'False' the roadblock is not removed

Specific cases

The roadblock may not be created. Possible causes are:
  • the agent doesn't have the physical capability.
  • the roadblock object definition is wrong (the object capacity "construction" for the 'roadblock' object is not set in the authoring tool).
If this case, the report 'Impossible work' is displayed, and the mission ends.

Comments

If needed, the agent dismounts to build the roadblock.

End of mission

This mission is permanent. It continues until another one is issued.
