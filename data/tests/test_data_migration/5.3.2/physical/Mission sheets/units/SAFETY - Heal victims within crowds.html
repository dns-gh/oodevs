Mission : SAFETY - Heal victims within crowds

Context

To bring immediate assistance and first aid on the spot in disasters.
In SWORD, to heal on the spot wounded victims within a set of crowds.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Crowds

The set of crowds to heal.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Behavior

The agent visits each crowd in the list (it moves to the center of the crowd concentrations). The order of the elements into the 'Crowds' list defines the order of visits.
Upon reaching a crowd, if it contains victims (wounded humans), the agent acts to heal it (the report "Starting to heal the crowd" is displayed). Once it is done (the report "wounded persons healed" should be displayed). As a consequence, the healed crowd no longer contains wounded humans.
The time required for healing does not depend on crowd size or the number of victims. Healing victims takes a few minutes (defined in decisional models).

Specific cases

Victims from a visited crowd may not be healed if:
  • the visited Crowd does not contain wounded humans ( the report "no wounded victims" and "victims cannot be evacuated" should be displayed)
  • the visited Crowd has moved (the agent won't follow the crowd) or the agent cannot reach it ( the report "victims cannot be evacuated" should be displayed) .
In this case, the agent decides to act on the next crowd.

Comments

End of mission

If there are no more crowds to visit, the mission ends.
