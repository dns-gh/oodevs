Mission : SAFETY - Evacuate urban blocks

Context

Emergency evacuation: removal of persons from a dangerous place due to a disaster or impending war.
In SWORD, acting on urban blocks to remove inhabitants.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Urban blocks

An ordered list of populated areas in which a population can be confined.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Behavior

The agent moves toward each provided urban block. The agent stops at the border of the urban block.
Upon reaching a urban block, the agent evacuates it before moving to the next unvisited urban block.
When all the urban blocks have been visited, the unit goes to the rally point.

Specific cases

An urban block may not be evacuated if:
  • the urban block cannot be inhabited (the report "urban block cannot be inhabited" is displayed)
  • the urban block has already been evacuated (the trace "the urban block is already evacuated" is displayed)
In this case, the agent decides to act on the next one.

Comments

End of mission

When the rally point is reached, the mission ends.
