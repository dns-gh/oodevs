Mission : LOG - Step up

Context

Set-up adopted by a unit for immediate or delayed action.
In sword, to provide new logistic devices to another logisitic company by reinforce it (the agent reinforces the company, meaning that the agent becomes subordinate to the new logistic base).

Parameters

Mandatory

Danger direction

Expected enemy direction.

Company

The company to reinforce.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Behavior
  • The agent moves to the company (it moves to the command post of the company). If it is not reachable (the command post is located into a urban block inside which the agent cannot move, for instance), the agent moves as close as possible. Average speed depends on the physical capabilities with respect to terrain properties.
  • Upon reaching the command post, the agent reinforces the company. As a consequence it becomes subordinate to the new company. The report "unit [ agentName ] has successfully reinforced [obectiveName ]" should be displayed.

Specific cases
  • If the company to renforce has moved, the agent won't follow it. To be reinforced, the agent needs to be nearby the command post (the interaction distance between the agent and the command post should be less than 100 METERS).

Comments
  • If needed, the agent dismounts.

End of mission

The terminates when the agent becomes subordinate to the provided company.
