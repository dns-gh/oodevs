Mission : LOG - Sort wounded

Context

In SWORD, for logistics agents, to know how to move toward a deployment position and self-deploy onto it. A deployed logistics agent enables its medical chains to sort wounded.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Deployment position

The position to reach.

Behavior
  • The agent self deploys onto its own position (the report "starting to deploy" then "section deployed" should be displayed). The deployment may take delays (see "Deployment" section in "Units" tab).
  • Upon deployment termination, the agent the activates its logistic sorting system.

Specific cases

A deployment position is provided:
  • The agent moves to the position. If the objective is not reachable, the agent moves as close as possible (the agent can move within it), the agent moves to the center, or to the closest position to the center). Average speed depends on physical capabilities, taking into accout terrain properties.
  • When the position has been reached, the agent deploys onto it (or nearby if it is not reachable) and activate the logistic sorting system.

Comments
  • If needed, the agent dismounts.

End of mission

This mission continues until another one is issued.
