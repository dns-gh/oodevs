Mission : ART - Deploy (on positions)

Context

Set-up adopted by a unit for immediate or later action.
In SWORD, to know how to move toward a deployment position (an object knowledge built by specific units or designated terrain location) and self deploy onto it. Once the agent gets deployed, it is ready to apply indirect fire (see fragmentary order "fire"). If a set of positions is issued, the agent can change its position after each fire execution.

Parameters

Mandatory

Danger direction

Expected ennemy direction.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Deployment positions

Set of positions to use to self deploy (default value is EMPTY).

Change position automatically

Define if the agent needs to move automatically to next position after each fire orders.

Behavior

If the deployment positions are not issued the agent deploys on its own position.
If positions are issued, the agent randomly chooses a position, moves toward it and self deploys onto it (the report "starting to deploy" then "section deployed" should be displayed). The deployment may take some time (see "deployment" section on "units" tab).
Upon deployment termination, the agent is ready to receive the fragmentary order "FIRE" to apply indirect fire (the report "ready to fire" is displayed).
Upon receiving the order to fire, the reports "Fire fragmentary order received" and "fire request received" are displayed and the agent launches the selected projectile.
If it cannot be possible (the agent has no more ammo for instance), the report "no ammo" is displayed.
In all cases, if the Change position automatically parameter is set to 'TRUE' (default value), the agent moves to a next position to avoid counter battery

Specific cases

The agent won't be able to open fire if:
  • the fragmentary order "FIRE" is issued while the agent is moving (from one position to another, for instance). The agent can receive the order only if it is deployed. If not the report "impossible to open fire, the shooter is not deployed" is displayed.
  • the agent doesn't have the physical capability (no suitable launcher for the dotation type): the report "No launcher with capacity to use specified ammunition" is displayed.
  • the agent does not have enough dotation: the report "no more shells" is displayed.
  • the selected resource type cannot allow the agent to perform an indirect fire : the report "Impossible indirect fire" is displayed.
If needed, the agent dismounts to deploy on each position.

Comments

End of mission

The mission ends if another mission (or "the cancel maneuver" fragmentary order) is issued by user.
