Mission : SAFETY - Lift alert on urban blocks within area

Context

To protect population from danger. Lifting an alert on a population is to inform it about the end of an impending danger.
In SWORD, a population is alerted via the urban block it populates. This mission is the reverse of the "Alert urban blocks within area" mission. Its goal is to lift an alert on a set of urban blocks within a specified zone.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Area

The area inside which urban blocks must be visited.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Behavior

The agent moves toward each urban block which is totally inside the specified area (the order of traversal is defined randomly). The agent stops at the border of the urban block.
Upon reaching an urban block, the agent lifts the alert on it before moving to the next unvisited urban block. The alert may not be lifted on the urban block if it cannot be populated (the report "no population in the urban block" is displayed).

Specific cases

Comments

End of mission

When there are no more urban blocks to visit, the agent stays next to the last visited block and the mission ends.
If there are no urban blocks within the area, the mission ends and the report "no urban blocks within area" is displayed.
