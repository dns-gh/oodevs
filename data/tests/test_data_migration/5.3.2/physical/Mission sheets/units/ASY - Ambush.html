Mission : ASY - Ambush

Context

The act of waiting in a concealed position in order to launch a surprise attack.
In sword, to move from one position to another in order to fire at enemies.

Parameters

Mandatory

Danger direction

Expected ennemy direction.

Positions

The positions to used to fire at enemies.

Meeting position

The position to reach when all positions have been used.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Enemies

The enemies to fire at.

Behavior
  • The agent moves to the first position and waits for enemies to come. If position is not reachable (an urban block inside which the agent cannot move, for instance), the agent moves as close as possible, otherwise it moves to the center of the position. Average speed depends on the physical capabilities regarding the terrain properties.
  • When a known enemy is approaching, the agent engages it once it is in range regarding the weapons system.
  • Upon firing action completion, the agent moves to the next position and so on until the last position is reached.
  • When firing on the last position, the agent moves to the meeting position.

Specific cases
  • The optional parameter 'enemies' can be filled if the ambush needs to be performed onto a specific enemy. In that case the agent engages only the provided enemies before moving to the next position.
