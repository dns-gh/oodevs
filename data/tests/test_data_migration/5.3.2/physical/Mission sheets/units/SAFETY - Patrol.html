Mission : SAFETY - Patrol

Context

Patrols are missions to gather information or to control rescue operations. Patrolling could ensure law, order and check crime in a given district.
In SWORD, patrolling consist of moving along a path defined by waypoints, indefinitely, allowing the agent to gather informations.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Waypoints

the path to move along.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Behavior
  • The agent moves toward each waypoints. The order of the elements into the 'Waypoints' list defines the order of visit. If a visited waypoint is not reachable, the agent moves as close as possible (the agents can move within it), the agent moves to the center, else it moves to a close position (the closest position from the center).
  • Upon reaching the last waypoints, the agent moves back to the first one and starts again.

Specific cases

Comments

End of mission

 This mission is permanent. It continues until another one is issued.
