Mission : COMMON - Move

Context

To move to an objective in order to reach it.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Objective

Objective to reach.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Behavior
  • The agent moves toward the objective. If the objective is not reachable, the agent moves as close as possible (the agent can move within it), the agent moves to the center, or to the closest position to the center).
  • Average speed depends on physical capabilities, taking into accout terrain properties.

Specific cases

A flying agent receiving this mission will firstly take off, then move towards the objective (just like a regular agent), then will land upon arrival to the objective.

Comments

End of mission

Upon reaching the objective (or a position nearby if it is not reachable), the trace "movement toward objective terminated" should be displayed and the mission terminates.
