Mission : ASY - Hide

Context

To hide in a given objective.
In Sword, for asymmetric threats (such as terrorists) to simply move to an objective trying to be invisible (the agent goes undetected).

Parameters

Mandatory

Danger direction

Expected enemy direction.

Objective

Objective to hide into

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Behavior
  • The agent moves toward the objective. If the objective is not reachable, the agent moves as close as possible (the agent can move within it), the agent moves to the center, else it moves to a close position (the position closest to the center).
  • Average speed depends on physical capabilities with respect to terrain properties.

Specific cases

Comments

End of mission

Upon reaching the objective the mission terminates.
