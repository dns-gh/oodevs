Mission : SAFETY - Operate a decontamination site

Context

To operate an NBC decontamination site in order to ensure units decontamination.
In SWORD, operating a decontamination site allows the agent to complete decontamination missions using that site. It decontaminates contaminated units (contaminated units may be issued the "go get decontaminated" mission).

Parameters

Mandatory

Danger direction

Expected enemy direction.

Decontamination site

the decontamination site object to be operated.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Behavior

The agent moves to the center of the object and starts to operate it (the report "Element is being operated" should be displayed).
The agent remains there until mission termination.

Specific cases

The decontamination site may not be operated if it cannot be operated (see object capacity "workable" in authoring tool), In that specific case, the report "element cannot be operated" is displayed and the mission ends.

Comments

Note that no specific capabilities are requeried for the agent to operate a decontamination site.

End of mission
