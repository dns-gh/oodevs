Mission : SAFETY - Patrol within area

Context

Patrols are missions to gather information or to control rescue operations.Patrolling could ensure law, order and check crime in a given district.
In SWORD, patrolling consist of moving along a path defined by waypoints, indefinitely, allowing the agent to gather informations. Here, waypoints are all reachable elements the agent can fin inside a given area

Parameters

Mandatory

Danger direction

Expected enemy direction.

Area

The area inside which the reachable elements need to be visited.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Behavior
  • The agent moves toward all elements that iare totally inside the specified area (the order of traversal is randomly defined). If a visited element is not reachable, the agent moves as close as possible (the agents can move within it), the agent moves to the center, else it moves to a close position (the closest position from the center).
  • Upon reaching an element, the agent get posted on its position (the report installation in progress" is displayed). Once it is done, the agent displays the report "installation completed" and moves to the next one.
  • Upon reaching the last element, the agent moves back to the first one and starts again.

Specific cases
  • If there is no reachable element within the area, the mission ends and the report "no key position within area" is displayed.
  • If the performing agent is flying (helicopters), it doesn't stop onto each position before moving to another one. The report "deployment in progress" and "installation completed" are not displayed.

Comments

Reachable elements are:
  • crossroads (point),
  • intersections between roads and the area limits,
  • urban blocks (those which are totally inside the specified area).

End of mission

The mission continues until another one is issued.
