Mission : ENGINEER - Resume construction works

Context

To resume construction of something.
In SWORD, to continue building a set of existing objects. This mission can be used to complete the construction of unfinished works. It could be used to model the repairing of damaged installations, urban blocks. It can be also used to complete the contruction and improvement of engineering obstacles (counter mobility support missions).

Parameters

Mandatory

Danger direction

Expected enemy direction.

Works

Meeting position

Position to reach when resumed work has terminated.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Wait for reinforcement

Define if agent needs to wait for reinforcements when it is not able to carry out work (false by default).

Behavior
  • The agent moves toward each work to construct. The order of visit is from the nearest to the furthest. The positions reached to construct each object are automatically determined by the agent.
  • Upon reaching a construction position, the agent starts to construct the object (the "work just started" report should be displayed). Upon construction termination (the report "work completed for object[object name]" should be displayed), the agent moves to the next work to do (next construction position for the next object to construct).
  • In obstacle contruction, if the parameter "Improve with mines" is set to TRUE, the agents improves (or completes the improvement of) each obstacle with mines.
  • Work may not be completed. If the agent doesn't have the physical capability or if it hasn't enough resources. In this specific case, the agent behaves in two ways depending on the "waitForReinforcement" parameter:
    • if it is set to FALSE, (default value) the report 'Impossible work' is displayed, and the agent moves to the next work to construct;
    • if it is set to TRUE, the agent will stay nearby the object and ask for reinforcements. It won't move to the next work until it is totally constructed.
  • If needed, the agent dismounts to construct an object.

Specific cases

Comments

End of mission

The mission is completed when all works have been visited; the agent moves to the meeting position and the mission terminates.
