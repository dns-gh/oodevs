Mission : ENGINEER - Reinforce

Context

In SWORD, a mission in which one unit augments the capability of another unit.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Unit

The element to reinforce (a unit or a company).

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Position

The position from which the agent will reinforce the objective (optional).

Behavior
  • The agent moves to the objective. If the position is not reachable (the objective is in a urban block inside which the agent cannot move, for instance), the agent moves as close as possible. Average speed depends on the physical capabilities with respect to terrain properties.
  • Upon reaching the objective, the agent reinforces it, only if the distance between the agent and the objective is less than 100 meters. The report "unit [ agentName ] has successfully reinforced [obectiveName ]" should be displayed.

Specific cases
  • If the parameter is an automat, the agent moves to the command post and becomes subordinated to the new company
  • If a position is issued, the agent moves to the position (not directly to the given agent or company). In that case it waits for the objective to be nearby before reinforcing it. The minimum distance should be less than 100 meters.

Comments

End of mission

The mission ends if another mission (or "the cancel maneuver" fragmentary order) is issued by user.
