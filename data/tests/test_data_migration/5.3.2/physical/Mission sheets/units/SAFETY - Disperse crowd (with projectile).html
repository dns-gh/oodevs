Mission : SAFETY - Disperse crowd (with projectile)

Context

To control, disperse, and arrest civilians who are involved in a riot, demonstration, or protest. Riot squads have long used less lethal weapons such as tear gas to disperse crowds and detain rioters.
In SWORD, a mission in which the agent launches a dispersion grenade that creates a specific object (a time limited object such as a crowd dispersion smoke object). Such an object is used to decreasethe number of healthy humans within the given crowd.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Crowd

The crowd to be dispersed.

Projectile

The type of resource (grenade) to use.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Position

The position from which the agent launches the grenade.

Behavior

The agent will try to launch the specified quantity of projectiles at the given crowd, from the issued position. If a position is not issued (default value), the agent moves toward the center of the crowd until it is at a good distance to launch the given projectile (the launching distance is defined by the distance between the agent's position and the center of the target crowd).
Upon reaching the position, the agent launches the projectile at the crowd. The mission then ends.

Specific cases

The agent may not be able to launch the projectile at the target crowd. Causes could be:
  • the agent doesn't have the physical capability (no suitable launcher with regard to dotation type): the report "No launcher with capacity to use specified munition" is displayed.
  • the agent does not have enough dotation: the report "no more shells" is displayed.
  • the selected resource type cannot allow the agent to perform indirect fire: the report "Impossible indirect fire" is displayed.

Comments

End of mission
