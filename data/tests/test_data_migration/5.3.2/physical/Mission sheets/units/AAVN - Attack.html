Mission : AAVN - Attack

Context

An offensive action characterized by movement supported by fire.
In SWORD, moving along a route defined by waypoints and engage all enemies in a provided area.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Route

The set of objectives to reach.

Area

The area inside of which the perceived enemies will be engaged.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Rally point

The rally point to go to when the last position of the route has been reached. If this parameter is not provided, then the agent will go to its initial position instead.

Aviation range

The aviation range enforced during the supplying.

Behavior
  • The agent moves toward the first objective. If it is not reachable (an urban block inside which the agent cannot move, for instance), the agent moves as close as possible, else it moves to the center of the objective. Average speed depends on the physical capabilities and terrain properties.
  • While moving toward the objective, the agent engages all enemies within the provided area it can.
  • Upon reaching the first objective the agent moves to the next one with the same behavior, until the last one is reached. Then, the agent will either move to the rally point if it is provided or move to its initial position.

The agent follows its itinerary, ignores the enemy outside the area, attacks the enemies inside the area, and goes back to its initial position (since no rally point was provided to the mission).

The agent follows its itinerary, attacks the enemies inside the area, and goes to the rally point.

Comments

End of mission

The mission ends when the rally point or the initial position of the agent is reached
