Mission : ENGINEER - Increase production

Context

To increase production of a resource network node.
In SWORD, to increase production of a resource network node.

Parameters

Mandatory

Danger direction

Resource network

Resource network node whose production is to be increased (several kinds of resource network node could be located in the same place: production can be increased for only one of them during the mission).

Optional

Phase lines

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Quantity (default value is 100)

Behavior
  • The agent moves toward the resource network node to increase production. The position of the resource network node is automatically determined by the agent.
  • If needed, the agent dismounts to reach the resource network node.
  • The agent tries to enter in the urban block, if the agent cannot reach the resource network node beacause of the trafficability of the urban block. The agent moves toward the border of the urban block.
  • Upon reaching the resource network node or the border of the urban block, the agent increases the production of the resource network node (the report "Setting up production for the resources network" should be displayed)

Specific cases

Comments

End of mission

The mission ends if another mission (or "the cancel maneuver" fragmentary order) is issued by user.
