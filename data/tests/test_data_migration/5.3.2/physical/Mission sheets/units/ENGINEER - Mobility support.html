Mission : ENGINEER - Mobility support

Context

The action of a force which aids, protects, complements, or sustains another force in accordance with a directive requiring such action.
  • A unit which helps another unit in battle. Aviation, artillery, or naval gunfire may be used as a support for infantry.
  • A part of any unit held back at the beginning of an attack as a reserve.
  • An element of a command which assists, protects, or supplies other forces in combat.
In SWORD, to perform engineer support in order to clear or bypass obstacles. The agent follows friend units to support, and remove blocking obstacles on demand.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Units

Friendly units to support.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Behavior
  • The agent follows friendly units (it follows the nearest unit) at a minimum distance (100 meters). If a chosen following position is not reachable (a followed agent stays in an urban block inside which the agent cannot move, for instance), the agent moves as close as possible (the agent may stay at a distance greater than 100 meters). Average speed depends on the physical capabilities with respect to terrain properties.
  • Upon reaching a supporting position, the agent is ready to intervene.
  • If followed units encounter obstacles on their computed path, they ask the agent to intevene. In this case, the agent moves to the obstacles and removes them. The first obstacle to be removed is the one that is the nearest to the unit.
  • If the supported unit can avoid the encountered obstacle by itself, a new path is computed and the unit continues on to the next waypoint using the new route, while the agent removes the obstacle.
