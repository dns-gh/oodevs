Mission : ASY - Harass a crowd

Context

In this Sword mission, to move to the given crowd and attack it in order to provoke people injuries.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Crowd

The 'Crowd' knowledge to attack.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Static attack

Defines if the agent needs to move onto random positions while attacking the provided crowd.

Behavior
  • The agent chooses a random position inside the nearest crowd's conentrations and moves to it. If the crowd's concentration is not reachable, the agent moves as close as possible (the agent cannot move within it).
  • Upon movement termination, the agent engages the crowd with lethal weapons systems.
  • When the agent has just engage the crowd, it chooses another position and starts again.

Specific cases
  • The crowd is moving:

    • The agent follows the head of the flow and contiunes to intervene on the crowd.
  • The 'static' parameter is set to 'TRUE'
    • the agent moves in the center of the crowd and stays here during confrontation.
  • If the target crowd is lost (the crowd knowledge becomes invalid) the mission terminates
  • The agent may self protect (against indirect fires or strong enemies) and stop temporarily the confrontation with the crowd.
Note that if the mission is performed by terrorists or militian , they won't selfprotect agaisnt strong enemies.
  • The crowd attacks the agent:

    • Crowds can attack agents of the simulation (e.g "Demonstrate mission with agressive attitude). In that case the agent selfprotect by moving a position outside the crowd.
    • User can issue the "stop safeguarding" fragmentary order to the agent. In that case, the agent keeps on harassing the crowd, even if it is assaulted. The agent may be injured.

Comments

The agent automatically sets its ROE against crowd with "lethal weapons authorized"

End of mission

The mission continues until another one is issued.
