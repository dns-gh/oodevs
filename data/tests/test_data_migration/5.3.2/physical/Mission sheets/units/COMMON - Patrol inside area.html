Mission : COMMON - Patrol inside area

Context

Patrols are missions to gather information or to control rescue operations. Patrolling could ensure law, order and check crime in a given district.
In SWORD, patrolling consists of moving along a path defined by waypoints, indefinitely, allowing the agent to gather information. Here, waypoints are automatically found inside the given area.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Area

The area in which the agent has to patrol.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Behavior
  • The agent moves toward each waypoint. The order of visit is randomly defined. If a visited waypoint is not reachable, the agent moves as close as possible (the agents can move within it), the agent moves to the center, otherwise it moves to a close position (the closest position to the center).
  • Upon reaching the last waypoint, the agent moves back to the first oneencountered and starts again.

Specific cases

If no position is found in the area (no road, no urban block), the agent moves to the center of the area.

Comments

Waypoints can be:
  • crossroads
  • urban blocks
  • points that intersect with the given area

End of mission
