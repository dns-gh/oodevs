Mission : AIR - Follow

Context

To follow an aircraft.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Aircraft

Aircraft to follow.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Height

Height to reach (in meters, default is 2000m).

Distance

The minimum distance between the agent and the aircraft to follow (in meters, default value is 500 meters).

Behavior

The agent takes off and then flies behind the given aicraft.

Specific cases

Comments

End of mission

The mission ends if another mission (or "the cancel maneuver" fragmentary order) is issued by user.
