Mission : LOG - Activate repair system

Context

Activating maintenance chain for repairing equipment.
In Sword, for logistics agents, to move toward a deployment position and self-deploy to it. A deployed logistics agent enables its logistics repair chains.

Parameters

Mandatory

Danger direction

Optional

Phase lines

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Deployment position

Position to reach to deploy to it (empty by default).

Behavior
  • If the Deployment position is not issued the agent deploys to its own position.
  • If a position is issued, the agent moves toward it and self-deploys to it (the report "starting to deploy" then "section deployed" should be displayed). The deployment may take some time (see "Deployment" section in "Units" tab).
  • Upon deployment termination, the agent activates its repair system chains.

Specific cases

Comments
  • If needed, the agent dismounts.

End of mission

The mission continues until another one is issued.
