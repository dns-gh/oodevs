Mission : COMBAT - Delay

Context

To delay enemy advance from successive positions
In Sword, to occupy successively a set of destroying positions in order to engage encountered enemies.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Positions

Ordered position to occupy.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Behavior
  • The agent reaches and occupies each of the issued positions. If a visited position is not reachable (an urban block inside which the agent cannot move, for instance), the agent moves as close as possible, else it moves to the center of the position.
  • Average speed depends on the physical capabilities regarding the terrain properties.
  • Upon reaching a position, the agent starts to observe and wait for enemies to come.
  • If enemies are detected, the agent engages them from the position it occupies. If the local force ratio is weak, the agent disengages and reaches the next position.

Specific cases

Comments

End of mission

The mission ends when the agent reaches the last position.
