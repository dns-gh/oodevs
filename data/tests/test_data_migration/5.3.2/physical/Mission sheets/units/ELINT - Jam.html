Mission : ELINT - Jam

Context

The deliberate radiation, reradiation, or reflection of electromagnetic energy to prevent or degrade the receipt of information by a receiver.
In Sword, to move toward an area and build a "jamming object" that fits with the area location (see definition of jamming effects in the authoring tool; 'Objects' tab, 'Capacities').

Parameters

Mandatory

Danger direction

Expected enemy direction.

Jamming object

Planned work to build by the agent.

Meeting position

Position to reach on "Disengage" fragOrder.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Behavior
  • The agent deploys to its own position. The time required for deployment depends on the data defined in the authoring tool (see "Units" tab, "Deployment" section). Upon deployment termination the report "section deployed" should be displayed.
  • Upon deployment termination, the agent activates the jamming by creating a "jamming area" object.

Specific cases
  • If the "disengage" order is issued, the agent moves to the meeting position (the jamming object is destroyed) and the mission ends.
  • If the "End mission" order is issued, the agent simply ends the mission (the jamming object is destroyed).
  • If another mission is issued, the agent performs the new mission and the jamming object is destroyed.

Comments

End of mission

The mission ends if another mission (or "the cancel maneuver" fragmentary order) is issued by user.
