Mission : COMBAT - Protect

Context

To protect a unit in combat or a humanitarian convoy.
In SWORD, a mission in which the unit augments the capability of the given unit. This mission is modeled with the reinforcement of a unit by another unit. A combatant agent augments the capability of another agent with its firing capabilities.
This mission is offensive. While protecting the unit, the agent that performs the mission will engage all perceived enemies in range.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Unit

The unit to reinforce with weapon capabilities.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Behavior
  • The agent moves to the unit (if it moves, the agent should follow it). If the position of the unit is not reachable (the unit is in a urban block inside which the agent cannot move, for instance), the agent moves as close as possible. The unit will be reinforced only when the distance between the agent and the unit is less than 100 meters). Average speed depends on the physical capabilities, taking into account terrain properties.
  • Upon reaching the unit, the agent reinforces it. The report "unit [ agentName ] has successfully reinforced [obectiveName ]" should be displayed. At this point the target unit is considered to be protected.

Specific cases

The agent cannot reach the unit: in this case the agent waits for the target unit to be reachable.

Comments

End of mission

The mission continues until another is issued.
