Mission : COMBAT - Escort prisoners to jail

Context

A member of the Armed Forces assigned to accompany, assist, or guide an individual or group.
An armed guard that accompanies a convoy, a train, prisoners, etc.
In SWORD, to accompany surrendered units toward a specific location (a position or a cam, for instance).

Parameters

Mandatory

Danger direction

Expected enemy direction.

Camp

Camp into which the agent has to accompany the surrendered units.

Surrendered units

Surrendered units to be transported to the given camp.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Behavior
  • The agent moves toward each of the given surrendered units, from the nearest to the furthest. If a unit position is not reachable (the unit is located inside an urban block inside which the agent cannot move, for instance), the agent moves as close as possible. Average speed depends on the physical capabilities regarding the terrain properties.
  • Upon reaching a surrendered unit, the agent loads it (magic tranportation).
  • Once all units have been loaded, they are transported to the given camp.
  • Upon reaching the camp (the agent moves to the center of it), all surrendered units are unloaded into the camp. They become prisoners and are taken into account by the logistics unit of the camp.

Specific cases
  • Note that if the issued object is not a camp (regarding the object's capacities in authoring tool), the mission ends with a failed status (the report "no camp available").
  • A prisoner unit may not be transported if it is not surrendered to the agent's side. In this case, the report "The unit is not surrendered" is displayed. If none of the issued agents is surrendered, the mission ends with a 'failed' status, displaying the report "No valid parameter".

Comments

End of mission

The mission ends once all the surrendered units have been jailed the camp.
