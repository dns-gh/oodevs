Mission : COMBAT - Block

Context

A mission assigned to a unit which requires it to deny enemy access to a given area or to prevent enemy advance in a given direction.
In Sword, to move and occupy an objective (or a set of objectives) and engage approaching enemies.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Objective

Objective to monitor.

Positions

Positions from which the subordinates observe the given objective.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Obstacles

Planned obstacles to errect.

Behavior
  • The agent chooses the best position to monitor the given objective. If one of the positions is not reachable (an urban block inside which the agent cannot move, for instance), the agent moves as close as possible, else it moves to the center of the position. Average speed depends on the physical capabilities with respect to terrain properties.
  • Upon reaching a position, the agent starts to observe the objective and waits for enemies to come.
  • If enemies are detected, the agent engages them from the position. Note that the agent may change its position while engaging the enemies.

Specific cases

Comments

End of mission

The mission ends if another mission (or "the cancel maneuver" fragmentary order) is issued by user.
