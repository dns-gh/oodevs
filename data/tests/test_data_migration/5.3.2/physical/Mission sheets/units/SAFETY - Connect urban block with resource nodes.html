Mission : SAFETY - Connect urban block with resource nodes

Context

In SWORD, connecting urban blocks with resources nodes allows an agent to create links from a distributing urban block to one or more receiving urban blocks for a given resource.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Urban block (producer)

The producer of the resource network to establish.

Resource nodes (receivers)

The list of the receivers of the resource network to establish, each item defined by a resource and the support of the node (object or urban block).

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Behavior

The agent moves toward each receivers in the order given by the operator. When the agent reaches a receiver, it establishes a resource network link from the producer to this receiver.

Specific cases

If the traficability of one of the receiving urban blocks is too low for the agent, it will report "Cannot reach objective" and skip the connection for this resource node.

Comments

End of mission

The mission ends when all of the receivers have been connected to the producer.
