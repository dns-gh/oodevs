Mission : COMBAT - Escort prisoners to position

Context

A member of the Armed Forces assigned to accompany, assist, or guide an individual or group.
An armed guard that accompanies a convoy, a train, prisoners, etc.
In SWORD, to accompany surrendered units toward a specific location (a position or a camp for instance).

Parameters

Mandatory

Danger direction

Expected enemy direction.

Position

Location into which the agent has to accompany the surrendered units.

Surrendered units

Units to be transported to the given location.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Behavior
  • The agent moves toward each given surrendered unit, from the nearest to the furthest. If a surrendered unit position is not reachable (the unit is located inside an urban block inside which the agent cannot move, for instance), the agent moves as close as possible. Average speed depends on the physical capabilities regarding the terrain properties.
  • Upon reaching a surrendered unit, the agent loads it ("magic/irrealistic" transportation).
  • Once all units have been loaded, they are transported to the given location.
  • Upon reaching the given position, all transported units are unloaded.

Specific cases
  • A prisoner unit may not be transported if it is not surrendered to the agent's side. In that case, the report "The unit is not surrendered" is displayed. If none of the issued agents is surrendered, the mission ends with a 'failed' status, displaying the report "No valid parameter".

Comments

End of mission

The mission ends once all the units have been transported to the given location.
