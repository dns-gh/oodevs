Mission : TRAFIC - Support unit movement

Context

All actions performed by the units in traffic control to support the projection by land forces and movements according to the priorities set by the command.
A unit which helps another unit to perform movement.
In SWORD, to perform the movement of unit by supporting it. The agent follows friend unit to support. The friend support unit speed increases to the maximum speed of the slowest equipment depends of the terrain if the support unit wasn't in this speed : in cover mode or inside town.

Parameters

Mandatory

Danger direction

Friend

Friend unit to support.

Optional

Phase lines

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Behavior

The agent moves toward the friend unit. The agent follows the friend unit. Average speed depends on the physical capabilities regarding to terrain properties.

Specific cases

Comments

End of mission

The mission ends if another mission (or "the cancel maneuver" fragmantary order) is issued by user.
