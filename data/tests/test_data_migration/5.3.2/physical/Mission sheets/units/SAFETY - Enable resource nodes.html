Mission : SAFETY - Enable resource nodes

Context

To provide basic needs and resources for populations, where production is insufficient to give an entire population a comfortable standard of living.
In SWORD, enabling a disabled node reactivates the transfer of resources between nodes. It can be used to help populations of urban blocks to be resupplied with resources after a terrorist attack (destruction of the node) or a preventive deactivation by authorities.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Resource nodes

The set of resource nodes to enable.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Behavior

The agent moves toward each of the resource nodes (a resource node is located within its corresponding urban block). If the urban block containing the resource node to enable is reachable (the agent can enter it), the agent moves to the center, otherwise it moves to a close position (the position closest to the center). The order of the elements the 'Resource nodes' list defines the order of visit.
Upon reaching a resource node (the urban block containing it), the agent enables it, displaying the report "Resource node enabled". The agent then decides to act on the next one.

Specific cases

Comments

End of mission

When there are no more resource nodes to enable, the agent stays next to the last visited node and the mission ends.
