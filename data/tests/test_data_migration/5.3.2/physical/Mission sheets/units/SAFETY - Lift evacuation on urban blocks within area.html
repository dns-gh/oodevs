Mission : SAFETY - Lift evacuation on urban blocks within area

Context

Emergency evacuation: removal of persons from a dangerous place due to a disaster or impending war.
In SWORD, a population is evacuated from the urban block it populates. This mission is the reverse of the "Lift evacuation from urban blocks within area" mission. Its goal is to lift an evacuation on a set of urban blocks within a specified zone.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Area

the area inside which urban blocks must be visited

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Behavior

The agent moves toward each urban block which is totally inside the specified area (the order of traversal is defined randomly). The agent stops at the border of the urban block.
Upon reaching an urban block, the agent lifts evacuation on it before moving to the next unvisited urban block.

Specific cases

The evacuation status may not be lifted if the urban block cannot be populated (the report "urban block cannot be inhabited" is displayed) . In this case, the agent decides to act on the block.

Comments

End of mission

When there are no more urban blocks to visit, the agent stays next to the last visited block and the mission ends.
If there are no urban blocks within the area, the mission ends and the report "no urban blocks within area" is displayed.
