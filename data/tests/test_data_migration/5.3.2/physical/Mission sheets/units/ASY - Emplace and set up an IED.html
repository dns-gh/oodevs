Mission : ASY - Emplace and set up an IED

Context

A surprise attack involving the deliberate use of violence against civilians in the hope of attaining political or religious aims.
In Sword, to use an agent's indirect fire capabilities in order to simulate an IED emplacement mission (performed by terrorists).

Parameters

Mandatory

Danger direction

Expected ennemy direction.

Emplacement

Location onto which the agent emplace the IED.

Hiding position

Position from which the agent activates the IED.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Automatic activation

Defines if the agent automatically activates the IED (if it needs explicit fragmentary order or not)(default value is false).

Behavior
  • The agent moves toward the emplacement. If it is not reachable (an urban block inside which the agent cannot move, for instance), the agent moves as close as possible, else it moves to the center of the objective. Average speed depends on the physical capabilities with respect to terrain properties.
  • Upon reaching the emplacement, the agent moves to the hiding position. Once it is reached, the agent is ready to activate the IED (the report "ready to activate IED" should be displayed) . At this stage, two possibilities:
    • the parameter 'Automatic activation' is set to 'true': the agent activates the IED.
    • the parameter 'Automatic activation' is set to 'false': the agent waits for the fragmentary order "Activate IED" to be issued.
  • Because the IED bomb is simulated with an indirect fire, the explosion can be delayed regarding the bomb activation. Because the explosion is resulting from an indirect fire, the reports "under artillery fire" and "friendly indirect fire" should be displayed too.
  • Upon bomb activation, all units around should be affected by the bomb. This terminates the mission.

Specific cases

The IED activation may fail if:
  • the distance between the hiding position and the IED emplacement is greater than the max range distance defined by the weapon systems (see authoring tool, weapon systems tab, "Hand and homemade bomb") . In this case the report "Cannot activate IED (target is out of range)" is displayed. This distance is usually set between 0 and 200 meters.
  • the agent doesn't have the adapted ammunition type or it hasn't enough of them (the report "no more shells" is displayed).
  • the fragmentary order is issued before the agent has reached the hiding position (the report "Cannot activate IED (IED has not been emplaced yet)" is displayed).
