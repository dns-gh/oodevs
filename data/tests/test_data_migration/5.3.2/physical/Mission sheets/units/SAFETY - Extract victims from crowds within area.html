Mission : SAFETY - Extract victims from crowds within area

Context

'To ensure evacuation of crowd victims, with suitable device.'
In SWORD, extracting victims from crowds allow the user to prepare the evacuation operation. In this mission, the agent extracts wounded and/or dead victims from all crowds located inside an area.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Area

The area inside which crowds must be visited.

Goal of the mission

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Reach crowds to extract victims

Behavior

The agent moves toward each crowd which is totally inside the specified area (the order of traversal is randomly defined).
Upon reaching a crowd, if it contains victims (wounded humans), the agent acts to extract victims (the report "starting to extract victims" is displayed). Once has finished (the report "victims extracted" should be displayed), a new crowd is created next to the initial crowd (with all wounded humans from initial Crowd). Note that the position of the newly created crowd is randomly determinated on the edge of the initial crowd. In consequence, the initial crowd no longer contains wounded humans.
The time taken for extraction does not depends on crowd size or the number of victims. Extracting victims takes a few minutes (defined in decisional models).

Specific cases

Victims of a visited crowd may not be extracted if:
  • the visited Crowd does not contain victims ( the report "no wounded victims" and "victims cannot be evacuated" should be displayed)
  • the visited Crowd has moved (the agent won't follow the crowd) or the agent cannot reach it ( the report "no wounded victims" and "victims cannot be evacuated" should be displayed)
In this case, the agent decides to act on the next crowd.

Comments

The type of victims that will be extracted from the crowds depends on the "Goal of the mission" parameter : it can either be wounded victims, dead victims, or both.

End of mission

If there is no crowd within the area, the mission ends and the report "no crowd within area" is displayed.
If there are no more crowsd to visit, the mission ends.
