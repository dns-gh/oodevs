Mission : COMBAT - Escort

Context

A member of the Armed Forces assigned to accompany, assist, or guide an individual or group. An armed guard that accompanies a convoy, a train, prisoners, etc.
In SWORD, to accompany a unit toward a specific location in order to protect it against potenital enemies or other dangers. The agent transports the unit to bring it to the given destination. No round trips here. The transportation is done by a magic action. The agent does not need specific physical capabilities.
This mission is offensive. While escorting the unit, the agent that performs the mission will engage all perceived enemies in range.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Unit

The unit to escort along the route.

Route

The route to move along. The last waypoint of the route defines the final objective.

Meeting position

The position to reach upon escort termination.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Behavior
  • The agent moves toward the given unit. If it is not reachable (the unit is located inside an urban block inside which the agent cannot move, for instance), the agent moves as close as possible. Average speed depends on the physical capabilities regarding the terrain properties.
  • Upon reaching the unit to escort, the agent loads it ("magic/irrealistic" tranportation).
  • Once the unit has been loaded, it is accompanied to the given destination.
  • Upon reaching the given destination, the transported unit is unloaded at the destination and the agent moves to the meeting position and the mission terminates.

Specific cases

Comments

End of mission

When the agents reaches the meeting position.
