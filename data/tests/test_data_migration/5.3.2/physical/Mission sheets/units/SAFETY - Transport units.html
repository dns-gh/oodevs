Mission : SAFETY - Transport units

Context

To ensure evacuation of victims, with adapted device toward adapted structure. e.g. to transport victims to an Hospital or to a place to care for victims.
In SWORD, to magically transport a set of units toward a specified destination. Transporting units allows to move them to a given location. Thus, a set of given victim units could be transported to a given position to receive care.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Units

the set of units to transport.

Target destination

the location to unload transported entities

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Behavior

The agent moves toward each entities to load (it moves to the exact position of the unit), by following the order of the user's choice to select the entities through the interface.
Upon reaching a unit, the agent loads it. The report "unit is loaded" should be displayed. Once all the units have been loaded, the agent moves to the target destination with the loaded units.
Upon reaching the target destination, the agent unloads one by one the transported units (for each of the unloaded unit, the report "unit is unloaded" is displayed).

Specific cases

A unit may not be loaded if :
  • it cannot be reached or if it has moved from its initial position ,
  • visited unit is already loaded or it is beeing transported.
In that specific case, the report "unit cannot be loaded" should be displayed.
If the destination location is not reachable, the agent moves as close as possible.

Comments

End of mission

The mission ends when all loaded units have been unloaded on the destination.
