Mission : AIR - Loiter

Context

For an aircraft to fly over objectives within area in order to gather information.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Route

Path to move along.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Height

Height to be reached by the agent (in meters, default value is 2000 meters).

Behavior
  • The agent takes off at the specified height and moves toward each waypoint defining the route.
  • Upon reaching a waypoint, the agent moves to the next one and so on, until the final one is reached.
  • Upon reaching the last waypoint the agent moves to the first one and moves again along the route.

Specific cases

Comments

End of mission

The mission ends if another mission (or "the cancel maneuver" fragmentary order) is issued by user.
