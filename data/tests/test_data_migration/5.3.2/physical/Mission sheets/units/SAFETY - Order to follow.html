Mission : SAFETY - Order to follow

Context

To visit a set of units and order them to follow

Parameters

Mandatory

Danger direction

Expected enemy direction.

Units

the set of teammate units to be asked to follow.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Behavior

The agent moves toward each target unit.
Upon reaching a unit, the agent asks it to follow. The report "Entity is requested to follow" should be displayed. The unit should start the "follow" mission and display the usual "follow" mission reports. If there is no more agent to visit, subject remains next to the last one visited.

Specific cases

A target unit may not follow the agent if it can't perform the "Follow" mission (the mission hasn't been set for the agent). The report should display the "Entity cannot follow (it can't perform the mission)". In that case the agent will go to next target.

Comments

End of mission

Followers will follow the agent until they receive another mission.
