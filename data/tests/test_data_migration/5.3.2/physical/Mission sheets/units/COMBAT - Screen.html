Mission : COMBAT - Screen

Context

Combat action which is to inform the ongoing progress of a running enemy in front of him with mobile elements which, without engaging or identifying, seizes any opportunity to clarify the information on the enemy.
In sword, to move from one position to another in order to monitor appoaching enemies.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Positions

The positions from which the agent monitors the approaching enemies.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right and left boundaries inside which the agent perform the mission.

Limit 2

Right and left boundaries inside which the agent performs the mission.

Stay on position at mission start

Defines if at the beginning of the mission, the agent may stay at its current position or if it may move directly to the first one.

Behavior
  • The agent moves to each position to monitor approaching enemies. If a visited position is not reachable (an urban block inside which the agent cannot move, for instance), the agent moves as close as possible, otherwise it moves to the center of the position.
  • Upon reaching a position,the agent waits for enemies to come.
  • The agent decides to move to the next one if its ratio of forces is low and if it has been detected by at least one enemy.

Specific cases
  • If the "disengage" fragmentary order is issued by user, the unit moves to the next position.
  • If the "Stay on position at mission start" is set to 'FALSE', the agent starts the mission by reaching the first position in the "positions" list.

Comments

The agent reaches the positions in the order of the 'positions' list.

End of mission

The mission ends if the agent reaches the last position or if another mission (or "the cancel maneuver" fragmentary order) is issued by user.
