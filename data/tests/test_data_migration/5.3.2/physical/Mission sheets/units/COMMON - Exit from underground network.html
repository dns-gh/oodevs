Mission : COMMON - Exit from underground network

Context

To exit from underground network (such as sewer, metro...) toward a given objective.
In Sword, to make the agent exit an underground network.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Objective

Final objective to reach.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Behavior
  • The agent simply gets out of the network (from its current occupied gateway) and then moves toward the given objective. If it is not reachable (the object is located inside an urban block withing which the agent cannot move), the agent moves as close as possible. Average speed depends on the physical capabilities while taking into account terrain properties.
  • Upon reaching the objective the mission terminates.

Specific cases

The agent won't be able to EXIT if:
  • the current underground exit is deactivated. In this case the agent waits until the exit is activated. Once this is done (with a magic frag order on the object, for instance) the agents gets out and finishes the mission in a normal way.
  • the current occupied gateway is destroyed (the object has disapeared): to get the agent out another gateway belonging to the network should be reached via the "Move within an underground network" mission. The a new mission "Exit from underground network" should then be issued.

Comments

For the agent, to exit from an underground network. This skill doesn't take an exit gateway as parameter. It only makes the agent go from the underground world to the outside world by moving toward a simple reachable objective.

End of mission

The mission ends once the objective is reached.
