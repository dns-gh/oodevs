Mission : COMMON - Request embark

Context

To be embarked on a position to be transported by other unit(s)
In Sword, to request the embarking of an agent by units on an embarking position. This mission is used to coordinate the troops and vehicules transportation (see transportation capabilities in authoring tool), when SWORD is connected to other simulations through an HLA federation.

Parameters

Mandatory

Danger direction

Embarking position

Embarking time

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Limit 2

Behavior
  • The agent moves toward the given position. If it is not reachable (a urban block inside which the agent cannot move for instance), the agent moves as close as possible, else it moves to the center of the position. Average speed depends on the physical capabilities regarding the terrain properties.
  • Upon reaching the position, the agent is ready to be embarked (the report "waiting for carriers" is displayed).

Specific cases

Comments

End of mission

The mission ends once the unit has been embarked by a carrier.
