Mission : ENGINEER - Activate obstacles

Context

To activate a set of reserved obstacles.
In SWORD, to activate a set of given objects representing engineer reserved obstacles.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Obstacles

Objects to activate.

Meeting position

Position to reach when task is finished.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Behavior
  • The agent moves toward each obstacle to activate. It acts on each obstacle by following the order of the elements in the 'Obstacles' list. The position to activate each obstacle is automatically determinated by the agent.
  • Upon reaching an obstacle, the agent starts to activate it. Upon work termination (the report "reserved obstacle activated [object name]" is displayed) the agent goes to the next one.
  • When all obstacles have been visited (even if a few of them couldn't be activated), the agent moves to the meeting position and the missions terminates.

Specific cases
  • An obstacle may not be activated. In this specific case, the report 'Obstacle [object name] cannot be activated' is displayed and the agent goes to the next obstacle to activate.

Comments

End of mission
