Mission : ART - Recon deployment sites

Context

The acquisition and processing of information concerning the enemy, the terrain, and the meteorological conditions; this information is necessary to make sure of the performance of combat activities of rocket troops and artillery.
In SWORD, to know how to reconnoiter positions and build a set of planned objects that may be used by ARTY firing units to get deployed. Thus, it is usually a mission preliminary to the artillery "deploy" mission.

Parameters

Mandatory

Danger direction

Expected ennemy direction.

Positions

Set of position to build (positions are OBJECTS here).

Meeting position

Meeting point to reach upon the reconnaissance is done.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Behavior

The agent moves toward each position to construct. The order of visit is user defined. The positions reached to build each position are automatically determined by the agent.
Upon reaching a position, the agent starts to reconnoiter it (it takes time) and then starts to construct the object position (the "work just started" report should be displayed).
Upon construction termination (the report "work completed for object[object name]" should be displayed), the agent moves to the next position.

Specific cases

Work may not be completed if the agent doesn't have the physical capability or lacks resources. In that case, the report 'work impossible, no capacity' is displayed, and the agent moves to the next position.
If needed, the agent dismounts to construct an position.

Comments

End of mission

The mission is finished when all positions have been visited. The agent then moves to the meeting position and the mission terminates.
