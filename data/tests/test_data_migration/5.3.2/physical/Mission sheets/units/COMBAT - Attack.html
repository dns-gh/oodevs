Mission : COMBAT - Attack

Context

An offensive action characterized by movement supported by fire.
In SWORD, moving along a route defined by waypoints and engage all enemies in range.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Route

The set of objectives to reach.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Behavior
  • The agent moves toward the first objective. If it is not reachable (an urban block inside which the agent cannot move, for instance), the agent moves as close as possible, else it moves to the center of the objective. Average speed depends on the physical capabilities and terrain properties.
  • While moving toward the objective, the agent engages all enemies within a distance that is sufficient regarding the weapon systems.
  • Upon reaching the first objective the agent moves to the next one with the same behavior, until the last one is reached.

Specific cases

Comments

End of mission

When the last objective has been reached.
