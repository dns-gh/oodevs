Mission : ENGINEER - Reconnoiter a crossing site

Context

A mission undertaken to obtain information by visual observation, or other detection methods, about the activities and resources of an enemy, or about the meteorological, hydrographic, or geographic characteristics of a particular area.
In SWORD, to move to a crossing site and reconnoiter it. Upon action termination, the agent gets information about the object (information is available through the knowledge of the crossing site object, such as water velocity, depth, width).

Parameters

Mandatory

Danger direction

Expected enemy direction.

Crossing site

The crossing site to reconnoiter.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Behavior

* The agent moves to the crossing site. If the crossing site is not reachable, the agent will move to the closest reachable position.

* Upon reaching the crossing site, the agent will start reconnoitering it (the report "Position reconnaissance started" should be displayed). After a delay depending on the size of the objective, the reconnaissance ends (the report "Position reconnaissance completed" should be displayed).

Specific cases

If the crossing site is not reachable, the agent will move to the closest reachable position. In this case, the agent will only reconnoiter it if it is close enough. If not, it will fail to reconnoiter, and the mission will end.

Comments

End of mission

The mission ends when the crossing site is reconnoitered, or if the agent finds that he cannot reconnoiter it.
