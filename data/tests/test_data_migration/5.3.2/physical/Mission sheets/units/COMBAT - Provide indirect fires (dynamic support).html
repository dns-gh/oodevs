Mission : COMBAT - Provide indirect fires (dynamic support)

Context

The action of a force which aids, protects, complements, or sustains another force in accordance with a directive requiring such action.
  • A unit which helps another unit in battle. Aviation, artillery, or naval gunfire may be used as a support for infantry.
  • A part of any unit held back at the beginning of an attack as a reserve.
  • An element of a command which assists, protects, or supplies other forces in combat.
In SWORD, supporting a unit is to provide intelligence and INDIRECT fires, in order to destroy or neutralize dangerous enemies.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Units

Friendly units to support.

Optional

Phase lines

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Retrograde context

Defines if the mission is performed in a retrograde context (default value is 'false').

Support only when units self-protect

Defines if the agent will support the provided units only if they self-protect. False by default.

Behavior
  • The agent chooses and moves toward a supporting position to get ready to provide fire for supported units. If the chosen position is not reachable (an urban block inside which the agent cannot move, for instance), the agent moves as close as possible, else it moves to the center of the position. Average speed depends on the physical capabilities, taking into account terrain properties.
  • The chosen position can change dynamically if the supported agents move. In that case, the agent moves from one position to another, trying to stay at a good range to provide indirect fire.
  • The agents only delivers indirect fire upon receiving an explicit 'fire' order from user. If such an order is issued (see 'fire' fragmentary order) the agent dismounts and launches the issued projectiles of chosen types.
  • In this mission, the agent does not need to self-deploy. It only needs to dismount (unlike an artillery mission).

Specific cases
  • Note that when the mission is performed by an agent which is subordinated to a commander agent (the company or squadron is ENGAGED), the targets are automatically retreived by the commander agent (they are agents that threaten forward units of the company) and the order to fire is automatically given once a unis is under enemy fire. The agent will move backwards, in the opposite direction as defined for the danger direction of the unit to support.

Comments

End of mission

The mission ends if another mission (or "the cancel maneuver" fragmentary order) is issued by user.
