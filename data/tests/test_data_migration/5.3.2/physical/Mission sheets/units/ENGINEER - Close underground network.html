Mission : ENGINEER - Close underground network

Context

To move to an underground entrance and close it...
In Sword, to reach an 'underground exit' object and deactivate it in order to prevent agents of simulation from using it (the agents won't be able to enter or exit the underground network using the supplied exit object).

Parameters

Mandatory

Danger direction

Expected enemy direction.

Underground exit

'Underground exit' object to close.

Meeting position

Position to reach upon action termination.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Behavior
  • The agent moves toward the given underground exit. If it is not reachable (the object is located inside an urban block within which the agent cannot move), the agent moves as close as possible. Average speed depends on the physical capabilities, taking into account terrain properties.
  • Upon reaching the underground exit, the agent simply deactivates it. The report "the underground exit has been deactivated" is displayed. The object status should switch from "activated: Yes" to "Activated: No" (see screenshot below).
  • Upon action termination, the agent moves to the meeting position.

Specific cases

Comments

End of mission

The meeting position is reached.
