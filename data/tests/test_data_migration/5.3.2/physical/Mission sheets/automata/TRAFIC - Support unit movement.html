Mission : TRAFIC - Support unit movement

Context

All actions performed by the units in traffic control to support the projection by land forces and movements according to the priorities set by the command.
A unit which helps another unit to perform movement.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Units or companies to support

Set of agents or companies to support.

Optional

Phase lines

Behavior
  • The commander agent issues the "Support unit movement" mission to the subordinate platoons, with the given agents or companies as parameter. Each platoon is given an agent to follow (the agent is randomly chosen by the commander agent).
  • Other units are ordered to reinforce or follow.

Specific cases

Comments

End of mission

The mission ends if another mission (or "the cancel maneuver" fragmantary order) is issued by user.
