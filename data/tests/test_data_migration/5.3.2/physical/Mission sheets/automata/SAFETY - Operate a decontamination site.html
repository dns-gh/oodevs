Mission : SAFETY - Operate a decontamination site

Context

To operate an NBC decontamination site in order to ensure units decontamination.
Operating a decontamination site allows the agent to complete decontamination missions using that site. It decontaminates contaminated units (contaminated units may be issued the "Get decontaminated" mission).

Parameters

Mandatory

Danger direction

Expected enemy direction.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Decontamination site

The decontamination site

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Behavior
  • The commander agent issues the "Operate" mission to the NBC platoons, with the site as parameter.
  • All the NBC units are ordered to move to the center of the objet and start to operate it if they can.
  • The "decontamination site" object define the maximum number of agent that can animate it. If more agent receive the mission, the will send a end mission when the arrived in the center of the object.
  • More agents operate the "decontamination site" object, quicker will be the decontamination.

Specific cases
  • Note that no specific capabilities are requeried for the subordinates units to operate a decontamination site.
  • Subordinates units cannot be contaminated by other units while operating a decontamination site.

Comments

End of mission

This mission continues until another one is issued.
