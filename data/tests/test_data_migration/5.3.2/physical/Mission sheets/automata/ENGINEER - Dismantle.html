Mission : ENGINEER - Dismantle

Context

To totally eliminate or neutralise a set of obstacles. A task that is usually performed by follow-on engineers and is not done under fire.
In SWORD, to remove a set of given Sword objects such as obstacles or human made infrastructure.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Objectives

Set of 'obstacle' objects to remove.

Meeting position

Position to reach by surbordinate units upon works termination.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Behavior
  • The commander agent issues the "Dismantle" mission to the engineer platoons, with a set of objectives as parameter. Objectives are fairly shared between engineer platoons.
  • Other units are ordered to reach the meeting position (if the number of engineer platoons is greater than the number of objectives, non used platoons are ordered to reach the meeting position).
  • An object may not be removed if one of the subordinate units doesn't have the physical capability or it hasn't enough resources to do it. In that case the company should be reinforced with adapted device.
  • When all objects have been removed, the engineer units move to the meeting position and the mission ends.

Specific cases
  • If an object is valorized with mines and if a subordinate unit that is performing the "Dismantle" mission cannot devalorize it (regarding its physical capacity), the commander agent will automatically reinforce it with another subordinate. If no subordinate unit can devalorize the obstacle, the commander agent waits to be reinforced with adapted devices.

Comments

End of mission

The mission terminates when all subordinates have finished their mission.
