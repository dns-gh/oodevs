Mission : COMBAT - Defend

Context

To hold a defined object against an enemy attack to halt or ward off an attack in order to defeat or destroy the enemy.
In Sword, to move and occupy an objective (or a set of objectives) and engage approaching enemies.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Objectives

Set of objectives to defend.

Meeting position

Position to reach by the command post and engineer units after performing their mission.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Obstacles

Set of obstacles ot errect by engineer units (no obstacle by default).

Echelon number

Define if the maneuver is organized into one or two echelons (default value is "2").

Behavior
  • Foward echelon units:
    • The commander agent issues the "Defend" mission to the teeth arm platoons, with the objectives as parameters,
    • The scouting platoons receive the "Monitor" mission with objectives and the main direction as parameters,
    • The engineer units receive the "Construct" mission if obstacles have to be erected (otherwise they receive the "follow" mission)
  • Rear echelon units:
    • Other units (reserve and non adapted units) receive the "follow" mission with the first echelon units as parameter.
    • In case of an attack by an enemy, combat units of the rear echelon may receive the "combat support" mission (the enemy may be engaged).
    • In case of a first echelon unit destruction, combat units of the rear echelon may relieve this unit.
  • All the agents share the same issued area of responsability.

Specific cases

If there are no engineer units, the teeth arm platoons will build the obstacles to be erected. Otherwise, only the engineer units will build the obstacles - unless some obstacles can not be built by the engineers, in which case the teeth arm platoons will try to build the remaining obstacles. In any case, the entities to build will be assigned equitably to the units.

Comments
The platoon is defending the objectives. There are no engineer units, therefore it is up to the teeth arm platoons to build the barricades.

Engineers construct the barricades, the mortars provide indirect fires to support all the other units.

End of mission

The mission ends if another mission (or "the cancel maneuver" fragmantary order) is issued by user.
