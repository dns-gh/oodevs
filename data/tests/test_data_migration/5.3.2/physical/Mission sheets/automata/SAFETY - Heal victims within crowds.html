Mission : SAFETY - Heal victims within crowds

Context

To bring immediate assistance and first aid on the spot in disasters.
In SWORD, to heal on the spot wounded victims within a set of crowds.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Crowds

The set of crowds to heal.

Rally point

Position to reach after performing the mission.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Behavior
  • The commander agent issues a "Heal victims within crowds" order to each capable agent. Every one of them is given a single crowd to heal victims from (the closest one to them), and whenever a subordinate has finished to process a crowd, another crowd is given to him, for as long as there are crowds left to process. When there are no longer such crowds, the subordinates are given a *Move To" order to the rally point instead.
  • The other units receive a *Move To* order to the rally point.

Specific cases

Victims from a visited crowd may not be healed if:
  • the visited Crowd does not contain wounded humans ( the report "no wounded victims" and "Wounded cannot be healed" should be displayed)
  • the visited Crowd has moved (the agent won't follow the crowd) or the agent cannot reach it .
In this case, the agent decides to act on the next crowd.

Comments

End of mission

The mission ends when all the subordinate agents have reached the rally point.
