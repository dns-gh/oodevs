Mission : ART - Deploy (on deployment sites)

Context

Set-up adopted by a unit for immediate ore later action.
In SWORD, to know how to order subordinate units to deploy onto a given set of deployment positions (objects knowledge built by specific units or designated terrain locations). Once the agents get deployed, they are ready to apply indirect fire (see fragmentary order "fire").

Parameters

Mandatory

Danger direction

Expected enemy direction.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Deployment positions

Set of positions to use to self deploy.

Behavior
  • The commander agent issues the "Deploy on positions" mission to the subordinate platoons, with a position as parameter. Each platoon is given a position (the position is randomly chosen by the commander agent).
  • Other units are ordered to follow.

Specific cases

Comments
  • To order deployed units to apply fire on chosen target, the user should disengage the commander agent and issues fragmentary order to each deployed unit.

End of mission
