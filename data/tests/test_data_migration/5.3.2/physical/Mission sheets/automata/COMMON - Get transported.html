Mission : COMMON - Get transported

Context

To move to a position in order to be transported by other unit(s)

Parameters

Mandatory

Danger direction

Expected enemy direction.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Position

Embarkation position.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Behavior
  • The commander agent issues a "getTransported" order to each subordinate with the position as parameter.

Specific cases

Comments

End of mission

The mission ends when all the subordinate agents have finished their mission (once they have been transported).
