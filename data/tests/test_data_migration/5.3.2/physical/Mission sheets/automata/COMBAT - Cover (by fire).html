Mission : COMBAT - Cover (by fire)

Context

A mission assigned to a unit which requires it to deny enemy access to a given area or to prevent enemy advance in a given direction.
In Sword, to move and occupy an objective (or a set of objectives) and engage approaching enemies.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Threats

Set of threat to monitor.

Positions

Set of positions from which the subordinate observe the given t

Entities

Set of entities to cover.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Behavior
  • Foward echelon units:
    • The commander agent issues the "Cover" mission to the armor platoons, with the threats and positions as parameters, (each of those parameters are shared between platoons).
    • The scouting platoons receive the "Monitor" mission with objectives and the main direction as parameters,
  • Rear echelon units:
    • Other units (reserve and non adapted units) receive the "follow" mission with the first echelon units as parameter.
  • In case of an attack by an enemy, combat units of the rear echelon may receive the "combat support" mission(the enemy may be engaged).
  • In case of a first echelon unit destruction, combat units of the rear echelon may releive this unit.
  • All the agents share the same issued area of responsability.
