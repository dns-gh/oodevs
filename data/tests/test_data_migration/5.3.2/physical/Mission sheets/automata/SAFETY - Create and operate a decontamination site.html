Mission : SAFETY - Create and operate a decontamination site

Context

To operate an NBC decontamination site in order to ensure units decontamination.
In SWORD, to build and operate a decontamination site object. Operating a decontamination site allows the agent to complete decontamination missions using that site. It decontaminates contaminated units (contaminated units may be issued the "Get decontaminated" mission).

Parameters

Mandatory

Danger direction

Expected enemy direction.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Decontamination site location

The NBC decontamination site to build and operate.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Remove installation (default value is TRUE)

Remove decontamination site at the end of the mission.

Behavior
  • The commander agent issues the "Create and Operate" mission to one of platoon (decontamination platoons), with the planned site as parameter.
  • Support units receive the "Follow and reinforce" mission with the NBC platoon as parameter.
  • The other units receive the "Move to" mission with the planned object as parameter.
  • When the site construction is completed, the NBC units move to the center of the objet and start to operate it.
  • Other units move to the center of the object and operate it if they have the capabilities.

Specific cases
  • The decontamination site may not be created if subordinate units do not have the capacity to build it regarding their physical capability and dotations. In that specific case, the report "work impossible, no capacity" is displayed by subordinate units.
  • The decontamination site may not be operated if it cannot be operated (see object capacity "workable" in authoring tool), In that specific case, the report "element cannot be operated" is displayed by subordinate units.
  • Note that no specific capabilities are requeried for the subordinates units to operate a decontamination site.
  • Subordinates units cannot be contaminated by other units while operating a decontamination site.

Comments

End of mission

This mission continues until another one is issued.
