Mission : ENGINEER - Bridge

Context

The purpose of any river crossing is to project combat power across a water obstacle to accomplish a mission. The challenge is to minimize the river's impact on the commander's ability to maneuver.
In Sword, to use given crossing sites object to construct a set of "bridge" objects in order to allow friend units to cross a water obstacle.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Meeting position

Position to reach by the command post and units that don't paracipate to the bridging operation.

Crossing sites

Set of objectives to defend.

Bridge system

Define if the bridge system is continuous or discontinuous.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Behavior
  • First stage: crossing sites reconnaissance.
    • The commander agent issues the "Reconnoiter a crossing site" mission to the engineer platoons, with a crossing site as parameter (one platoon per crossing site to reconnoiter).
    • Other units are ordered to follow.
  • Second stage : the commander agent issues the "Bridge" mission to the engineer platoons, with a crossing site as parameter (one platoon per crossing site to bridge),
    • Other units stays on the meeting position.
    • Once the "bridge" objects are constructed, friends units should be able to cross the water obstacle.

Specific cases

Comments
  • For the commander agent, to order subordinates units to bridge in order to make friend units cross a water obstacle. The mission execution defines two stages: first one is the reconnaissance of the crossing sites, the second one is the bridging of the crossing sites.
  • Note that only engineer units work for that mission. Other units reach the meeting position.

End of mission

The mission ends if another mission (or "the cancel maneuver" fragmantary order) is issued by user.
