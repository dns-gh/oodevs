Mission : ART - Reconnoiter and deploy (on deployment sites)

Context

The acquisition and processing of information concerning the enemy, the terrain, and the meteorological conditions; this information is necessary to make sure of the performance of combat activities of rocket troops and artillery.
In SWORD, to order subordinate reconnaissance units to build a set of planned objects (that represent "reconnoitered artillery positions") and then to order firing subordinate units to deploy onto them (to get ready to provide indirect fires)

Parameters

Mandatory

Danger direction

Expected enemy direction.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Positions

Set of 'position' objects to reconnoiter and to build (positions are specific objects to build in order to be used by ARTY firing units) .

Meeting position

Position to reach by the command post and units that don't paracipate to the bridging operation.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Change positions automatically

Behavior
  • The commander agent issues the "ReconnoiterDeploymentSites" mission to the reconnaissance platoons, with the set of positions as parameter. Upon reconnaissance termination the 'position' objects are created. Each positions are fairly shared between reconnaissance platoons.
  • Other units are ordered to reach the meeting position.
  • A position may not be built if the subordinate reconnaissance platoon doesn't have the physical capability or it hasn't enough resources to do it. In that case, the unit goes to next position.
  • When all positions have been reconnoitered and built, the reconnaissance units move to the meeting position and the commander waits the order "continue mission" to be issued by user (the report "waiting order deploy firing units" is displayed).
  • Once the fragmentary order is issued, the commander order firing subordinate units to deploy onto created 'position' objects. One units for one position.
  • Other units are ordered to reach the meeting position.

Specific cases

Comments
  • To order deployed units to apply fire on chosen target, the user should disengage the commander agent and issues fragmentary order to each deployed unit.

End of mission
