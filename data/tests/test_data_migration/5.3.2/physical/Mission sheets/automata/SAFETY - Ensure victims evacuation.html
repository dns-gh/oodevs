Mission : SAFETY - Ensure victims evacuation

Context

To ensure evacuation of victims, with suitable device.
In SWORD, to evacuate victim units (such as refugee units, for instance), using a specified evacuation unit (a medical unit linked to the medical chain).

Parameters

Mandatory

Danger direction

Expected enemy direction.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Victim units

The team-mate agents inside which victims must be evacuated with the medical unit.

Medical unit (log)

The team-mate agent linked to the medical chain.

Rally point

Position to reach after performing the mission.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Behavior
  • The commander agent issues a "Ensure victims evacuation" order to each capable agent. Every one of them is given a single unit to evacuate (the closest one to them), and whenever a subordinate has finished to evacuate a unit, another unit to evacuate is given to him, for as long as there are units left to evacuate. When there are no longer such units, the subordinates are given a *Move To* order to the rally point instead.

  • The other units receive a *Move To* order to the rally point.

Specific cases

The agent may not be able to ensure the wounded are evacuated. Causes could be:
  • the agent doesn't have the physical capability (the agent IS NOT defined as a medical unit)
  • the medical chain is not activated for the 'Medical unit'
  • the visited 'Victims unit' does not contain wounded humans
  • the visited 'Victims unit' has moved or the agent cannot reach it
If those cases are encountered, the agent moves to the next 'Victim unit', and the report "victims cannot be evacuated" should be displayed.
If the medical unit is not a logistic unit, the victim will not be evacuated, and the report "The medical unit is not a logistic unit" should be displayed.

Comments

End of mission

The mission ends when all the subordinate agents have reached the rally point.
