Mission : SAFETY - MoveTo

Context

To move to an objective using a defined route.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Waypoints

The waypoints to reach.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Behavior
  • The commander agent issues a "moveTo" order to each subordinate with the waypoints as parameter.

Every agent moves to the waypoints.

Specific cases

If one of the waypoints is outside of the limits, the subordinates will move to the closest position inside the limits, then move on to the next waypoint (except if it the last waypoint, in which case the mission ends).

The agents will go to the position inside the limits the nearest of the first waypoint.

The agents will then go to the next waypoint as usual.

Comments

End of mission

The mission ends when all the subordinate agents have finished their mission.
