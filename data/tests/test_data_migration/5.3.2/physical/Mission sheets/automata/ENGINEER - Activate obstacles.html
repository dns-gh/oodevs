Mission : ENGINEER - Activate obstacles

Context

To activate a set of reserved obstacles.
In SWORD, to know how to activate a set of given objects representing engineer reserved obstacles.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Obstacles

Set of 'obstacle' objects to activate.

Meeting position

Position to reach by the command post and non-used units.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Behavior
  • The commander agent issues the "Activate obstacles" mission to the engineer platoons, with an obstacle as parameter. Obstacles are fairly shared between engineer platoons.
  • Other units are ordered to reach the meeting position (if the number of engineer platoons is greater than the number of obstacles, non-used platoons are ordered to reach the meeting position).
  • When all obstacles have been activated, the engineer units move to the meeting position.

Specific cases

Comments

End of mission

The mission ends when all the subordinate agents have finished their mission.
