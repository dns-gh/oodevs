Mission : SAFETY - Operate a checkpoint (filter crowds)

Context

In SWORD, checkpoints are object that can filter crowds. Using such an object allows the agent to filter crowds in order to detect hidden units (terrorist or other characters) and prevents them from going into restricted areas.

Parameters

Mandatory

Danger direction

Expected enemy direction.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Checkpoint

The object to be used to filter crowds.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Intensity (%, 100 by default)

The intensity to apply to filter crowds, 100% by default.

Behavior
  • The commander agent issues the "Operate" mission to the platoons, with the checkpoint as parameter.
  • All the units are ordered to move to the center of the objet and start to operate it if they can.

Specific cases
  • Note that no specific capabilities are requeried for the subordinates units to operate a checkpoint.

Comments

End of mission

This mission continues until another one is issued.
