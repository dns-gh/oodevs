Mission : COMMON - Move to in column formation

Context

To move to an objective in order to reach it.

Parameters

Mandatory

Danger direction

Expected ennemy direction.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Destination

Destination to reach.

Distance

The minimum distance to respect between agents.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Behavior

* The commander agent issues a "moveTo" order to the CP agent.
* The commander agent issues a "follow" order to each subordinate with a différent agent to follow.
* The minimal distance is respected between each agent.

Specific cases

Comments

End of mission

The mission ends if another mission (or "the cancel maneuver" fragmantary order) is issued by user.
