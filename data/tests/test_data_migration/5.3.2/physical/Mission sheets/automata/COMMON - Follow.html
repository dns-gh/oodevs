Mission : COMMON - Follow

Context

To follow a set of motile agents (units or crowds).

Parameters

Mandatory

Danger direction

Expected enemy direction.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Agents

Set of entities to follow.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Behavior
  • The commander agent issues a "follow" order to each subordinate with the set of "Agents" as parameter.

Specific cases

Comments

End of mission

The mission ends if another mission (or "the cancel maneuver" fragmantary order) is issued by user.
