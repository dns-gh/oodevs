Mission : COMBAT - Escort prisoners to position

Context

A member of the Armed Forces assigned to accompany, assist, or guide an individual or group. An armed guard that accompanies a convoy, a train, prisoners, etc.
In SWORD, to accompany a refugees or surrendered units toward a specific location (a position or a camp for instance).

Parameters

Mandatory

Danger direction

Expected enemy direction.

Limit 1

Right or left boundaries inside which the agent perform the mission.

Limit 2

Right or left boundaries inside which the agent perform the mission.

Location

Location into which the agent has to accompany the surrendered units.

Surrendered units

Units to be transported to the given location.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Behavior
  • Forward echelon units :
    • The commander agent issues the "EscortPrisonersToPosition" mission to each of the first echelon units,
    • The number of first echelon units depends on the number of surrendered units to escort (one surrendered unit corresponds to one surbodinate unit).
  • Rear echelon units:
    • The commander agent issues the "follow" mission with the first echelon units as parameter.
    • In case of an attack by an enemy or bad force ratio, combat units of the second echelon may receive the "combat support" mission (the enemy may be engaged).

Specific cases

Comments

End of mission

The mission ends when all surrendered units have been unloaded onto the issued location.
