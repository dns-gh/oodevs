Mission : COMMON - Move along in column formation

Context

To move to an objective in order to reach it.

Parameters

Mandatory

Danger direction

Expected ennemy direction.

Limit 1

Right or left boundaries inside which the agent performs the mission.

Limit 2

Right or left boundaries inside which the agent performs the mission.

Route

Path to move along.

Optional

Phase lines

Lines used by units to adapt their behavior when they reach or cross them.

Distance

The minimum distance to respect between agents.

Behavior
  • The commander agent computes the smallest maximal speed among all the units in the company. The commander agent then issues a *Move To* order to every subordinate (except the CP unit) towards positions around the CP unit at a certain distance (this distance matches the distance that a unit can cover when going at the previously computed speed).
  • Once every subordinate is on its assigned position, the commander agent will issue a "Move To" order to the CP agent with the provided waypoints, and then at each time step, another subordinate will receive *Move To" order with the provided waypoints but starting with the initial position of the CP agent.

The company agent computes the initial positions.

The subordinates reach these positions.

The CP agent starts its movement.

A subordinate will receive the *Move To* order, then another one, and so on.

The company moves along the path in a column formation.

Specific cases

A flying agent receiving this mission will take off at the beginning of the mission, and will land upon arrival at the final objective.

Comments

End of mission

The mission ends if another mission (or "the cancel maneuver" fragmantary order) is issued by user.
