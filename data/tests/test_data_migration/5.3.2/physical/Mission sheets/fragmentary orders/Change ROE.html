Mission : Change ROE

Context

To order the agent to change its rules of engagement.

Parameters

Mandatory

ROE type

  • Free fire: the agent is free to open fire to any enemi unit, depending on its mission. This rule doesn't mean that the agent automatically open fire when an enemy is in range. This should happen only through the mission the agent is performing (attack or recce in force for instance).
  • Retaliation only: the agent can only fire back to units that have previously open fire on it.
  • Fire upon order: the agent never open fire, even if it is attacked.
