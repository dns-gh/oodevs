Mission : Fire

Context

To apply indirect fire(s) on a selected target.
In SWORD, a fragmentary order in which the agent launches a projectile to deliver indirect fire.

Parameters

Mandatory

Ammunitions

Type of ammunitions to use.

Shells number

Quantity of projectile to use (empty by default).

Target

Location onto which the projectile must be launched.

Optional

Behavior

Specific cases
  • The agent won't be able to open fire if :
    • the agent doesn't have the physical capability (no adapted launcher regarding to dotation type).
    • the agent does not have enough dotation.
    • the selected resource type cannot allow the agent to perform an indirect fire.

Comments

End of mission
