Mission : Go to (at all cost)

Context

For the agent to reach a given destination at ALL COST. This order will make tha agent moves to the destination even if it is performing a mission.

Parameters

Mandatory

Objective

Objective to reach at all cost.

Optional

Behavior

This order is not permanent: upon reaching the objective, the task terminates and the agent moves on with its current mission.

Specific cases

Comments

End of mission
